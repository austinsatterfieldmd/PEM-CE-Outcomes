## ⚠️ CRITICAL: Non-Oncology Question Detection

**BEFORE tagging, verify this is truly an ONCOLOGY question.** Questions that MENTION CMV may actually be about a different specialty.

### Secondary Immunodeficiency (SID) Questions — Flag as NON-ONCOLOGY

Patients with hematologic malignancies and cancer often develop secondary immunodeficiency. Questions about **managing infections or diagnosing immunodeficiency** belong to **Infectious Disease/Immunology**, NOT oncology.

**Red flags that suggest NON-ONCOLOGY (SID):**
1. Activity name contains "SID", "Secondary Immunodeficiency", or "Immunodeficiency"
2. Patient has CMV BUT the question asks about:
   - Infection diagnosis/management (CMV, PJP, fungal, etc.)
   - Immunoglobulin replacement therapy
   - Vaccination in immunocompromised patients
   - Tests to diagnose immunodeficiency (not cancer)
3. Answer choices focus on immunology/infectious disease concepts, NOT cancer treatment

**If you detect this pattern:**
- Tag `topic: "Non-oncology - SID"` to flag for manual review
- The question will be removed from the oncology dataset and reclassified

**True oncology questions in infection context:**
- "How should you modify treatment during active infection?" → ONCOLOGY (about CMV treatment modification)
- "Which CMV treatment has highest infection risk?" → ONCOLOGY (about drug safety)

## CMV Clinical Context

CMV is a major complication in:
- **Allogeneic stem cell transplant (allo-HSCT)** - Most common setting
- **Solid organ transplant (SOT)** - Lung, heart, kidney, liver
- **CAR-T cell therapy** - Emerging concern
- **Immunocompromised patients** - HIV/AIDS, immunosuppressive therapy

### CMV Clinical Scenarios

| Scenario | Description |
|----------|-------------|
| **CMV prophylaxis** | Prevention in high-risk patients BEFORE reactivation |
| **Preemptive therapy** | Treatment triggered by viremia detection (PCR+) before disease |
| **CMV disease/infection** | Active symptomatic CMV (pneumonitis, colitis, retinitis) |
| **Refractory/Resistant CMV** | CMV not responding to first-line therapy |

**NOTE:** This table lists COMMON drugs for reference. It is NOT exhaustive. ### Primary Prophylaxis Agents

| Drug | Context | Notes |
|------|---------|-------|
| **Letermovir** | Post-allo-HSCT prophylaxis | CMV terminase inhibitor, FDA approved 2017 |
| **Maribavir** | Treatment (not prophylaxis) | UL97 kinase inhibitor |
| **Ganciclovir/Valganciclovir** | Prophylaxis AND treatment | Myelosuppressive |
| **Acyclovir/Valacyclovir** | Limited CMV activity | Not recommended for CMV-specific prophylaxis |

### Treatment Agents

| Drug | Context | Notes |
|------|---------|-------|
| **Ganciclovir (IV)** | First-line treatment | Myelosuppression is dose-limiting |
| **Valganciclovir (oral)** | Oral prodrug of ganciclovir | For less severe disease or step-down |
| **Foscarnet** | Second-line, ganciclovir-resistant | Nephrotoxicity, electrolyte abnormalities |
| **Cidofovir** | Third-line | Significant nephrotoxicity |
| **Maribavir** | R/R CMV infection | 2021 FDA approval for refractory CMV |

### Key CMV Trials

| Trial | Drug | Context |
|-------|------|---------|
| **SOLSTICE** | Maribavir | R/R CMV post-transplant |
| **Letermovir prophylaxis trials** | Letermovir | CMV prophylaxis post-allo-HSCT |

### answer_length_pattern Criteria (CRITICAL - Common Error)

**DEFAULT: If `distractor_homogeneity` = "Homogeneous", ALWAYS tag `answer_length_pattern` = "Uniform".**

Override to "Variable" or "Correct longest/shortest" ONLY when the length difference is EGREGIOUS — e.g., one answer is a single word and another is a full sentence with additional clinical detail.

Do NOT tag "Variable" just because:
- One option has a few extra words
- Compound answers are longer ("Both A and B" vs "A only")
- Drug name lengths differ

**When in doubt with Homogeneous distractors → tag "Uniform".**

Tag "Variable" with Homogeneous distractors ONLY when one answer contains **additional clinical/educational content** the others lack - NOT when length differs due to:
- Grammatical phrasing ("The drug ganciclovir" vs "Ganciclovir")
- Compound answers ("Both A and B", "A and C", "All of the above")
- Minor word count differences

**"Uniform"** = All options are CONCEPTUALLY similar in complexity:
- All single drug names: "Ganciclovir" / "Foscarnet" / "Maribavir" → **Uniform**
- All short clinical phrases: "Start prophylaxis" / "Monitor viral load" / "Treat preemptively" → **Uniform**

**"Variable"** = Options differ in TYPE or COMPLEXITY AND contain different amounts of clinical detail:
- Mix of single words and detailed explanations → **Variable**
- One answer has extra clinical context others lack → **Variable**

**"Correct longest/shortest"** = The CORRECT answer is noticeably longer/shorter than ALL distractors:
- Correct answer is a detailed explanation while distractors are single words → **Correct longest**
- Correct answer is a single drug while distractors are multi-drug regimens → **Correct shortest**

### 62. distractor_homogeneity

### distractor_homogeneity Criteria (CRITICAL - Common Error)

**"Homogeneous"** = All distractors are the SAME TYPE of answer:
- All drug names: "Ganciclovir" / "Foscarnet" / "Maribavir" / "Letermovir" → **Homogeneous**
- All clinical actions: "Hold therapy" / "Dose reduce" / "Discontinue" / "Switch drug" → **Homogeneous**
- Mix of single drugs and combos is still **Homogeneous** (all are treatment options)

**CRITICAL:** Implausible distractors are still the same TYPE. Tag `flaw_implausible_distractor: true` but keep `distractor_homogeneity: "Homogeneous"` if all options are drugs/treatments.

**"Heterogeneous"** = Distractors are DIFFERENT TYPES of answers:
- Treatment + Safety + Mechanism: "Ganciclovir" / "Neutropenia occurs in 30%" / "DNA polymerase inhibition" → **Heterogeneous**
- Drug + Monitoring + Endpoint: "Maribavir" / "Weekly CMV PCR" / "Viral clearance" → **Heterogeneous**
