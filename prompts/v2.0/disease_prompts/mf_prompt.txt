## ⚠️ CRITICAL: Non-Oncology Question Detection

**BEFORE tagging, verify this is truly an ONCOLOGY question.** Questions that MENTION MF may actually be about a different specialty.

### Secondary Immunodeficiency (SID) Questions — Flag as NON-ONCOLOGY

Patients with hematologic malignancies and cancer often develop secondary immunodeficiency. Questions about **managing infections or diagnosing immunodeficiency** belong to **Infectious Disease/Immunology**, NOT oncology.

**Red flags that suggest NON-ONCOLOGY (SID):**
1. Activity name contains "SID", "Secondary Immunodeficiency", or "Immunodeficiency"
2. Patient has MF BUT the question asks about:
   - Infection diagnosis/management (CMV, PJP, fungal, etc.)
   - Immunoglobulin replacement therapy
   - Vaccination in immunocompromised patients
   - Tests to diagnose immunodeficiency (not cancer)
3. Answer choices focus on immunology/infectious disease concepts, NOT cancer treatment

**If you detect this pattern:**
- Tag `topic: "Non-oncology - SID"` to flag for manual review
- The question will be removed from the oncology dataset and reclassified

**True oncology questions in infection context:**
- "How should you modify treatment during active infection?" → ONCOLOGY (about MF treatment modification)
- "Which MF treatment has highest infection risk?" → ONCOLOGY (about drug safety)

## CRITICAL: Compound Answer Handling

When the correct answer references options by letter/number (e.g., "B or C", "A and C", "Options 1 and 3"):

1. **DO NOT assume option order matches the original question** - The options you receive may be in a different order than the actual test question.

2. **Deduce the correct answer from clinical reasoning** - Read the substance of each option and determine which ones make sense as the correct answer based on the clinical scenario.

3. **Tag based on the deduced correct content** - Once you identify what the compound answer actually refers to, tag those treatments/concepts.

**Example (MF):**
- **Stem:** 65-year-old with intermediate-2 risk MF, splenomegaly, constitutional symptoms, JAK2 V617F positive, platelet count 120,000. What is the most appropriate treatment?
- **Correct answer:** "A or B"
- **Options presented:** Ruxolitinib | Fedratinib | Hydroxyurea | Watch and wait

**How to deduce:** The patient has symptomatic intermediate-2 risk MF with JAK2 mutation and adequate platelets. Therefore "A or B" most likely refers to:
- Ruxolitinib (first-line JAK inhibitor)
- Fedratinib (alternative JAK inhibitor, especially if anemia concern)

**Tag accordingly:** treatment_1: Ruxolitinib, treatment_2: Fedratinib

## MF DRUG LANDSCAPE (CRITICAL)

**NOTE:** This table lists COMMON drugs for reference. It is NOT exhaustive. If a question mentions a drug, drug class, or mechanism not listed here, STILL tag it using the appropriate fields (treatment_1-5, drug_class_1-3, drug_target_1-3). Always tag what the question discusses, even if not in this table.

### JAK Inhibitors (Primary Treatment Class)
**Target:** JAK1/JAK2 signaling pathway

| Drug | Status | Context | Key Differentiator |
|------|--------|---------|-------------------|
| **Ruxolitinib** | FDA approved (2011) | First-line for intermediate/high-risk | Standard first-line JAK inhibitor |
| **Fedratinib** | FDA approved (2019) | First-line OR post-ruxolitinib | Can use in JAK inhibitor-naive or post-ruxolitinib |
| **Pacritinib** | FDA approved (2022) | Severe thrombocytopenia (platelets <50K) | Only JAK inhibitor for severe thrombocytopenia |
| **Momelotinib** | FDA approved (2023) | Symptomatic anemia + splenomegaly | Unique anemia benefit (ACVR1 inhibition) |

**KEY DIFFERENTIATORS:**
- **Ruxolitinib**: First-line standard, requires platelets >=50K for full dosing
- **Fedratinib**: Alternative first-line, also approved post-ruxolitinib
- **Pacritinib**: ONLY option for platelets <50K (PERSIST trials)
- **Momelotinib**: Best for anemia + splenomegaly (inhibits ACVR1/hepcidin)

**INFERENCE:**
- Symptomatic MF, intermediate/high-risk, adequate platelets -> Ruxolitinib (first-line)
- Post-ruxolitinib failure -> Fedratinib, momelotinib, or pacritinib
- Platelets <50,000 -> Pacritinib
- Significant anemia + splenomegaly -> Consider momelotinib
- Transfusion-dependent anemia -> Consider momelotinib

### Allogeneic Stem Cell Transplant
- Only potentially curative treatment for MF
- Reserved for higher-risk patients (Int-2, High)
- Age and fitness determine eligibility

### Supportive/Other Therapies (DO NOT TAG unless specifically asked about)
- Hydroxyurea (cytoreduction)
- Danazol (for anemia)
- ESAs (erythropoiesis-stimulating agents)
- Transfusions
- **Exception:** Tag if question specifically asks about these agents

### answer_length_pattern Criteria (CRITICAL - Common Error)

**DEFAULT: If `distractor_homogeneity` = "Homogeneous", ALWAYS tag `answer_length_pattern` = "Uniform".**

Override to "Variable" or "Correct longest/shortest" ONLY when the length difference is EGREGIOUS — e.g., one answer is a single word and another is a full sentence with additional clinical detail.

Do NOT tag "Variable" just because:
- One option has a few extra words
- Compound answers are longer ("Both A and B" vs "A only")
- Drug name lengths differ

**When in doubt with Homogeneous distractors → tag "Uniform".**

Tag "Variable" with Homogeneous distractors ONLY when one answer contains **additional clinical/educational content** the others lack - NOT when length differs due to:
- Grammatical phrasing ("The drug ruxolitinib" vs "Ruxolitinib")
- Compound answers ("Both A and B", "A and C", "All of the above")
- Minor word count differences

**Example - UNIFORM (despite length difference):**
- "Ruxolitinib only"
- "Fedratinib only"
- "Both ruxolitinib and fedratinib"
-> Third answer is longer but just combines options, no new clinical detail = **Uniform**

**"Uniform"** = All options are CONCEPTUALLY similar in complexity:
- All single drug names: "Ruxolitinib" / "Fedratinib" / "Pacritinib" -> **Uniform**
- All short clinical phrases: "Continue therapy" / "Switch treatment" / "Refer for transplant" -> **Uniform**

**"Variable"** = Options differ in TYPE or COMPLEXITY AND contain different amounts of clinical detail:
- Mix of single words and detailed explanations: "Ruxolitinib" / "Continue hydroxyurea and monitor for 8 weeks before reassessing response" -> **Variable**

**"Correct longest/shortest"** = The CORRECT answer is noticeably longer/shorter than ALL distractors

### distractor_homogeneity Criteria (CRITICAL - Common Error)

**"Homogeneous"** = All distractors are the SAME TYPE of answer:
- All drug names: "Ruxolitinib" / "Fedratinib" / "Pacritinib" / "Momelotinib" -> **Homogeneous**
- All clinical actions: "Hold therapy" / "Dose reduce" / "Discontinue" / "Continue" -> **Homogeneous**
- All trial names: "COMFORT-I" / "JAKARTA" / "PERSIST-2" / "MOMENTUM" -> **Homogeneous**

**CRITICAL:** Implausible distractors are still the same TYPE. Tag `flaw_implausible_distractor: true` but keep `distractor_homogeneity: "Homogeneous"` if all options are drugs/treatments.

**"Heterogeneous"** = Distractors are DIFFERENT TYPES of answers:
- Treatment + Safety + Mechanism: "Ruxolitinib" / "Anemia occurs in 40%" / "JAK1/2 inhibition" -> **Heterogeneous**
- Drug + Trial + Endpoint: "Fedratinib" / "JAKARTA study" / "Spleen response" -> **Heterogeneous**

|-----------------|-------|
| Intermediate-2 or High-risk MF, symptomatic | -> JAK inhibitor indicated |
| First-line, adequate platelets | -> Ruxolitinib or fedratinib |
| Platelets <50,000 | -> Pacritinib (ONLY option) |
| Post-ruxolitinib + anemia | -> Consider momelotinib |
| Post-ruxolitinib + adequate platelets | -> Fedratinib or momelotinib |
| Transfusion-dependent anemia | -> Consider momelotinib |
| Young + high-risk | -> Consider transplant |
| COMFORT trial | -> Ruxolitinib, JAK inhibitor-naive |
| JAKARTA trial | -> Fedratinib |
| PERSIST trial | -> Pacritinib, thrombocytopenia |
| MOMENTUM trial | -> Momelotinib, post-ruxolitinib |

---

## Important Guidelines for MF

1. **Risk stratification drives treatment** - Low/Int-1 often watch and wait; Int-2/High need treatment
2. **Platelet count is CRITICAL** - <50K = pacritinib only; >=50K = other JAK inhibitors
3. **Anemia burden matters** - Momelotinib has unique anemia benefit
4. **Post-ruxolitinib is common context** - Know second-line options
5. **Transplant is only cure** - But limited to younger, fit, higher-risk patients
6. **Driver mutations matter** - JAK2/CALR/MPL for diagnosis, HMR mutations for prognosis
7. **Tag from CORRECT answer only** - Don't tag distractors
