## disease_type: Subtyping Hierarchy

**CRITICAL PRIORITY:** Always try very hard to determine the molecular subtype. Subtype determination is the #1 priority for breast cancer.

### Step 1: Determine HER2 Status First
```
HER2 IHC 3+ OR IHC 2+/ISH+ → HER2-positive (HER2+)
HER2 IHC 1+ OR IHC 2+/ISH- → HER2-low
HER2 IHC 0 with membrane staining → HER2-ultralow
HER2 IHC 0 OR negative → HER2-negative
```

### Step 2: Determine Final Subtype

| Scenario | disease_type Value |
|----------|-------------------|
| HER2+ and HR+ both explicit | `"HR+/HER2+"` |
| HER2+ (HR- or not specified) | `"HER2+"` |
| HER2-low (any HR status) | `"HER2-low"` |
| HER2-ultralow | `"HER2-ultralow"` |
| HR+ and HER2-negative | `"HR+/HER2-"` |
| ER-, PR-, HER2- | `"Triple-negative"` |
| Ductal carcinoma in situ | `"DCIS"` |
| Invasive lobular carcinoma focus | `"ILC"` |

### Edge Case: HR+/HER2-low Classification
**Critical decision for Treatment selection questions:**
- If answer involves **T-DXd** (trastuzumab deruxtecan) → Use `"HER2-low"` or `"HER2-ultralow"`
- Otherwise → Use `"HR+/HER2-"` (CDK4/6i, endocrine therapy drive treatment)
- *Rationale:* T-DXd is the only drug specifically approved for HER2-low. When T-DXd is the treatment, HER2-low status is clinically actionable.

### Edge Case: HER2-ultralow and T-DXd (DESTINY-Breast06)
**HER2-ultralow** is a newly defined category based on DESTINY-Breast06 (DB-06) trial findings:
- **Definition:** HER2 IHC 0 with **faint, incomplete membrane staining** in ≤10% of cells — previously grouped with HER2-negative
- **Clinical significance:** DB-06 showed T-DXd benefit in HER2-ultralow patients, expanding treatment eligibility beyond HER2-low (IHC 1+ or IHC 2+/ISH-)
- **When to use `"HER2-ultralow"`:**
  - Question discusses T-DXd in patients with IHC 0 with faint staining
  - Question references DB-06 / DESTINY-Breast06 specifically
  - Question asks about reclassification of HER2 IHC 0 specimens
  - Question discusses the distinction between "true HER2 0" (no staining) and "HER2-ultralow" (faint staining)
- **IHC scoring hierarchy for disease_type:**
  - IHC 3+ or IHC 2+/ISH+ → `"HER2+"` (always)
  - IHC 2+/ISH- or IHC 1+ → `"HER2-low"`
  - IHC 0 with faint membrane staining → `"HER2-ultralow"`
  - IHC 0 with NO staining → HER2-negative (use `"HR+/HER2-"` or `"Triple-negative"` based on HR status)

## treatment_line: Breast Cancer Conventions

**Early-Stage:**
- `"Adjuvant"` - After surgery
- `"Neoadjuvant"` - Before surgery
- `"Perioperative"` - Before and after surgery

**Metastatic:**
- `"1L"` - First-line, treatment-naive metastatic
- `"2L"` - Second-line (after progression on one prior line)
- `"3L"` - Third-line (after progression on two prior lines)
- `"4L+"` - Fourth-line or later
- `"2L+"` - Second-line or later, **use ONLY when the exact line cannot be determined** (e.g., stem says "heavily pretreated" or "multiple prior lines" without listing them)
- `"Maintenance"` - Continuation after response

**Specificity rule:** When prior therapies are explicitly listed in the stem and you can count the lines, use the specific value ("2L", "3L", "4L+"). Fall back to "2L+" only when the exact count is ambiguous.
- Example: "treated with trastuzumab + pertuzumab and trastuzumab emtansine" = 2 prior lines → current treatment is `"3L"`
- Example: "after progression on first-line CDK4/6i + AI" = 1 prior line → current treatment is `"2L"`
- Example: "heavily pretreated, multiple prior lines" → `"2L+"` (exact count unknown)

- `"ATM"`, `"CHEK2"` - moderate-penetrance genes (genetic risk counseling)
- `"TP53"` - Li-Fraumeni syndrome, genetic risk
- `"MSI-H"` / `"TMB-high"` - rare but actionable (pembrolizumab)

**Genomic Assays — tag as biomarker when the assay/score is discussed:**

**Oncotype DX (21-gene recurrence score):**
- `"Oncotype DX"` - GENERIC: Use when the question is about the test itself (what it measures, when to order, clinical utility)
- `"Oncotype DX RS-high"` - When the question specifies or implies a HIGH recurrence score (RS ≥26) driving a treatment decision (chemotherapy benefit)
- `"Oncotype DX RS-low"` - When the question specifies or implies a LOW recurrence score (RS ≤25) and the decision is to omit chemotherapy
- `"Oncotype DX RS-intermediate"` - When the question specifically discusses the intermediate range (RS 16-25) and age-based decision-making (TAILORx/RxPONDER context)
- **IMPORTANT:** When a specific Oncotype DX score is given in the stem (e.g., "Oncotype Dx = 10"), ALWAYS tag the RS category, not the generic assay name. Score of 10 → `"Oncotype DX RS-low"`. Score of 30 → `"Oncotype DX RS-high"`.
- Key trials: TAILORx (node-negative), RxPONDER (1-3 positive nodes)

**MammaPrint (70-gene signature):**
- `"MammaPrint"` - GENERIC: Use when the question is about the test itself
- `"MammaPrint high-risk"` - When clinical significance of high genomic risk is discussed
- `"MammaPrint low-risk"` - When low genomic risk drives omission of chemotherapy
- Key trial: MINDACT (can identify clinical high-risk/genomic low-risk patients who can safely omit chemo)

**Breast Cancer Index (BCI):**
- `"BCI"` - GENERIC: Use when the question is about the test itself
- `"BCI-high"` - When BCI result drives decision for EXTENDED adjuvant endocrine therapy (beyond 5 years)
- `"BCI-low"` - When BCI result supports stopping endocrine therapy at 5 years
- BCI is unique: it predicts benefit of EXTENDED endocrine therapy (years 5-10), not chemotherapy

**Prosigna / PAM50 (risk of recurrence score + intrinsic subtype):**
- `"Prosigna"` - GENERIC: Use when discussing the test
- Provides both a risk of recurrence (ROR) score AND intrinsic subtype (Luminal A/B, HER2-enriched, basal-like)
- Key context: Used in node-negative and node-positive early BC for prognosis

**EndoPredict (EPclin):**
- `"EndoPredict"` - 12-gene molecular score combined with clinical features (tumor size, nodal status)
- Predicts late recurrence risk (years 5-15), helps guide extended endocrine therapy decisions

**Other genomic/molecular assays:**
- `"FoundationOne CDx"` - Comprehensive genomic profiling (CGP) — used for identifying actionable mutations (PIK3CA, ESR1, BRCA, TMB)
- `"Guardant360"` - Liquid biopsy CGP — ctDNA-based mutation detection
- `"Signatera"` - Personalized ctDNA MRD (minimal residual disease) assay for recurrence monitoring

**Ki-67 — Use specific values when clinical significance is discussed:**
- `"Ki-67"` - GENERIC: Use when the question is about Ki-67 as a test (what it measures, interpretation, IHC methodology)
- `"Ki-67-high"` - When the question states or implies HIGH Ki-67 (≥20-30%) driving a treatment decision (e.g., CDK4/6i adjuvant eligibility in monarchE, Luminal B classification)
- `"Ki-67-low"` - When the question states or implies LOW Ki-67 (<20%) suggesting favorable biology (Luminal A, may omit chemotherapy)

**ctDNA/Liquid Biopsy — tag when discussed:**
- `"ctDNA"` - circulating tumor DNA (ESR1 mutation detection, minimal residual disease, treatment monitoring)

**NORMALIZATION REMINDER:** In clinical vignettes, always normalize mutation biomarkers:
- Tag `"PIK3CA mutation"` (not `"PIK3CA"`) when the question discusses PIK3CA mutation status
- Tag `"ESR1 mutation"` (not `"ESR1"`) when the question discusses ESR1 mutation status
- These normalized forms ensure consistent downstream analysis

**CRITICAL: Where to tag genomic assay results:**
- Tag the ASSAY NAME in `biomarker_1-5` fields (e.g., `"Oncotype DX RS-high"`)
- Tag the CLINICAL IMPLICATION in `disease_specific_factor` when it drives staging/risk (e.g., `"High genomic risk"`, `"Low genomic risk"`)
- Do NOT duplicate: if Oncotype DX RS-high is in biomarker, do not also put "Oncotype DX" generically

## Breast Cancer Topic-Specific Guidance

### Surgical Management Questions
**Tag as topic: `"Surgical management"`** for questions about:
- **Breast surgery type:** Breast-conserving surgery (BCS/lumpectomy) vs mastectomy (total, modified radical, skin-sparing, nipple-sparing)
- **Axillary surgery:** Sentinel lymph node biopsy (SLNB) vs axillary lymph node dissection (ALND), targeted axillary dissection (TAD), number of nodes for staging
- **Surgical margins:** Re-excision criteria, margin width definitions, "no ink on tumor"
- **Reconstruction:** Implant-based (prepectoral vs submuscular/subpectoral), autologous (DIEP flap, TRAM flap, latissimus dorsi), timing (immediate vs delayed), oncoplastic surgery
- **Anesthesia for breast surgery:** PECS I/II blocks, paravertebral blocks, enhanced recovery protocols
- **Surgical timing:** Surgery relative to neoadjuvant therapy, interval to adjuvant treatment
- **Benign/borderline tumors:** Phyllodes tumor excision (wide local excision, margin requirements), fibroadenoma management, desmoid fibromatosis resection
- **Contralateral prophylactic mastectomy:** Risk reduction in BRCA carriers, bilateral mastectomy decisions

**Tagging rules for Surgical management:**
- treatment_1-5: Tag the SURGICAL PROCEDURE when it is the focus (e.g., `"mastectomy"`, `"lumpectomy"`, `"SLNB"`, `"ALND"`, `"DIEP flap"`)
- If the question also discusses adjuvant systemic therapy, tag the systemic drug in treatment fields and use topic: `"Treatment selection"` (not Surgical management)
- drug_class for surgical questions: typically `null` unless a systemic agent is also discussed

### Radiation Therapy Management Questions
**Tag as topic: `"Radiation therapy management"`** for questions about:
- **Post-BCS radiation:** Whole breast irradiation (WBI), hypofractionated WBI, boost to tumor bed
- **Post-mastectomy radiation (PMRT):** Indications (T3/T4, ≥4 positive nodes, positive margins), chest wall + regional nodal fields
- **Regional nodal irradiation (RNI):** Axillary, supraclavicular, internal mammary node coverage
- **Accelerated partial breast irradiation (APBI):** Eligibility criteria, techniques (brachytherapy, IORT, external beam)
- **Dose/fractionation:** Conventional (50 Gy/25 fx), hypofractionated (40-42.5 Gy/15-16 fx), ultra-hypofractionated (26 Gy/5 fx per FAST-Forward)
- **Radiation side effects:** Skin toxicity, pneumonitis, cardiac risk (left-sided), lymphedema, rib fractures
- **Omission of radiation:** Low-risk elderly patients (PRIME II, LUMINA criteria), DCIS management
- **Radiation + systemic therapy sequencing:** Concurrent vs sequential with chemotherapy, timing with immunotherapy

**Tagging rules for Radiation therapy management:**
- treatment_1-5: Tag `"radiation therapy"` or specific technique (e.g., `"APBI"`, `"PMRT"`) when radiation is the focus
- If the question is about radiation as part of a multimodal approach where systemic therapy is the focus, use the systemic therapy topic instead

### Pathological Classification Questions
**Tag as topic: `"Pathological classification"`** for questions about:
- **Histologic typing:** IDC (no special type/NST), ILC, tubular, mucinous, papillary, medullary-like, micropapillary, metaplastic, adenoid cystic
- **Grading:** Nottingham histologic grade (tubule formation + nuclear pleomorphism + mitotic count), Bloom-Richardson equivalent
- **Molecular subtyping:** Luminal A, Luminal B, HER2-enriched, basal-like, normal-like (PAM50/intrinsic subtypes)
- **Ki-67 interpretation:** Cutoff values, prognostic significance, role in treatment decisions
- **Special pathologic features:** Lymphovascular invasion (LVI), tumor-infiltrating lymphocytes (TILs), perineural invasion, extensive intraductal component (EIC)
- **pCR assessment:** Definition of pathologic complete response (ypT0/is ypN0), residual cancer burden (RCB) scoring
- **Pre-malignant lesions:** ADH, ALH, flat epithelial atypia (FEA), LCIS (classic, pleomorphic, florid), DCIS (low/intermediate/high grade, comedo vs non-comedo)
- **Staging pathology:** TNM staging components, number of positive nodes, extranodal extension

### Screening and Prevention Questions
**Tag as topic: `"Screening and prevention"`** for questions about:
- **Risk assessment models:** Tyrer-Cuzick (IBIS), Gail model, BRCAPRO, CanRisk — when to use each, inputs, limitations
- **Screening guidelines:** Mammography initiation/frequency (ACR, ACS, USPSTF), supplemental screening with MRI (high-risk), DBT, contrast-enhanced mammography
- **Genetic testing/counseling:** BRCA1/2, PALB2, ATM, CHEK2, TP53, germline panel testing candidacy, cascade testing, VUS interpretation
- **Preventive therapy:** Tamoxifen (pre/postmenopausal), raloxifene (postmenopausal), aromatase inhibitors (postmenopausal high-risk), exemestane (MAP.3 trial)
- **Risk-reducing surgery:** Prophylactic bilateral mastectomy, bilateral salpingo-oophorectomy (BSO), timing in BRCA carriers
- **High-risk management:** Enhanced surveillance protocols, breast MRI alternating with mammography every 6 months

### Imaging Questions
**Tag as topic: `"Imaging"`** for questions about:
- **Diagnostic imaging:** Mammography (diagnostic vs screening), ultrasound (solid vs cystic), breast MRI (indications: high-risk screening, extent of disease, ILC workup, occult primary)
- **Staging imaging:** CT chest/abdomen/pelvis, bone scan, FDG PET/CT (when to use, false positives)
- **Novel imaging:** FES PET (fluoroestradiol — for ER+ disease, heterogeneity assessment), contrast-enhanced mammography (CEM), molecular breast imaging (MBI)
  - **FES F18 PET tagging:** When a question discusses FES PET / FES F18 PET imaging, ALWAYS tag `"FES F18 PET"` in a biomarker field (biomarker_1-5) in addition to setting topic to "Imaging". FES PET is a functional imaging biomarker for ER expression.
- **Surveillance imaging:** Post-treatment follow-up, when to image vs clinical monitoring
- **BI-RADS classification:** Category interpretation (0-6), management recommendations per category

### Survivorship Questions
**Tag as topic: `"Survivorship"`** for questions about:
- **Surveillance strategies:** Follow-up schedule, when to discontinue mammography, role of tumor markers (CA 15-3, CA 27.29 — limited utility)
- **Cardiotoxicity:** Anthracycline-related (cumulative, irreversible), trastuzumab-related (reversible, LVEF monitoring), radiation-associated cardiac disease
- **Bone health:** AI-associated bone loss, DEXA screening, bisphosphonates/denosumab for bone protection, adjuvant bone-modifying agents
- **Fertility preservation:** Pre-treatment options (embryo/oocyte cryopreservation, GnRH agonist ovarian suppression), post-treatment fertility, pregnancy after breast cancer
- **Sexual health/GSM:** Genitourinary syndrome of menopause (vaginal dryness, dyspareunia), vaginal estrogen safety in HR+ disease, non-hormonal options (ospemifene, fezolinetant for vasomotor symptoms)
- **Neurotoxicity:** Chemotherapy-induced peripheral neuropathy (CIPN) — taxane-related, management (duloxetine), cognitive impairment ("chemo brain")
- **Lymphedema:** Risk factors, prevention, early detection, management
- **Psychosocial:** Distress screening, body image, return to work, cancer-related fatigue

## Special Cases

### BIA-ALCL (Breast Implant-Associated ALCL)
- disease_state: `"Breast cancer"` (classified under breast for our system)
- topic: `"Surgical management"` (complication requiring surgical management — implant removal + capsulectomy)
- disease_type: `null` (not a standard breast cancer subtype)
- toxicity_type_1: `"BIA-ALCL"` (tag as a toxicity/complication of breast implants)
- Tag implant removal procedure in treatment fields if discussed

### DCIS
- disease_type: `"DCIS"`
- disease_stage: `"Early-stage"` (always — stage 0)
- Questions about DCIS management may be topic: `"Surgical management"` (excision decisions), `"Radiation therapy management"` (post-lumpectomy RT), or `"Treatment selection"` (endocrine therapy for risk reduction)

### Benign Breast Conditions (Still Tagged as Breast Cancer)
These conditions should still use disease_state: `"Breast cancer"` but with appropriate disease_type:
- **Phyllodes tumor:** disease_type: `"Phyllodes"`, topic typically `"Surgical management"`
- **Desmoid fibromatosis:** disease_type: `"Desmoid"`, topic typically `"Surgical management"`
- **ADH/ALH:** disease_type: `"ADH"`, topic typically `"Pathological classification"` or `"Screening and prevention"`
- **LCIS:** disease_type: `"LCIS"`, topic varies (pathology, screening, surgical)
- **Fibroadenoma:** disease_type: `null`, topic typically `"Diagnosis"` or `"Surgical management"`

### Pregnancy and Breast Cancer
- special_population_1: `"Pregnant"`
- Treatment modifications: Avoid trastuzumab, taxanes generally safe in 2nd/3rd trimester, no endocrine therapy during pregnancy
- If question is about treatment during pregnancy → topic: `"Treatment selection"` with special_population tagged

### Supportive Care in Breast Cancer Context
Questions about supportive care (anti-emetics, pain, CIPN, vasomotor symptoms) that are in a breast cancer CE activity:
- Tag disease_state: `"Breast cancer"` (it's within a breast cancer educational context)
- Topic: `"AE management"` (if managing a treatment side effect) or `"Survivorship"` (if about long-term symptom management)
- Tag the supportive care drug in treatment fields (e.g., `"fezolinetant"`, `"duloxetine"`, `"olanzapine"`)

### Treatment Inference for AE Management and Survivorship Topics
**The same treatment-inference logic that applies to Treatment selection also applies to AE management and Survivorship questions.** When a patient is described as being ON a treatment and experiencing side effects, tag those causative treatments in the treatment fields.

- Example: "Premenopausal patient on OFS (GnRH agonist) + AI complaining of joint pain and hot flashes" → tag `treatment_1: "Goserelin"`, `treatment_2: "Leuprolide"` (GnRH agonists), `drug_class_1: "GnRH agonist"`, `drug_class_2: "Aromatase inhibitor"`
- Example: "Patient on T-DXd develops ILD" → tag `treatment_1: "Trastuzumab deruxtecan"`, `drug_class_1: "Antibody-drug conjugate (ADC)"`
- Do NOT leave treatment fields blank just because the topic is AE management — the causative treatment is clinically relevant

# Field Definitions (66 LLM-Tagged Fields)

You must assign values for all 66 fields below. Use `null` for fields that are not applicable or cannot be determined.

## Value Selection Guidance

**Strict fields** (marked with "Valid values"): Use ONLY the listed values.
- topic, disease_stage, treatment_line, toxicity_organ, toxicity_grade, outcome_context, clinical_benefit, guideline_source, evidence_type, cme_outcome_level, data_response_type, stem_type, lead_in_type, answer_format, answer_length_pattern, distractor_homogeneity

**Open fields with common values** (marked with "Common values"): Prefer the listed values when they apply. If the question contains a value not in the list but you are confident it is correct, you may use that value instead.
- disease_state, disease_type, treatment_1-5, biomarker_1-5, trial_1-5, drug_class_1-3, drug_target_1-3, prior_therapy_1-3, resistance_mechanism, metastatic_site_1-3, symptom_1-3, toxicity_type_1-5, efficacy_endpoint_1-3, disease_specific_factor, comorbidity_1-3

---

**CRITICAL: Only tag comorbidities explicitly stated in the question stem.** Comorbidities are medical conditions SEPARATE from the primary disease. Do NOT infer comorbidities from age, disease stage, or treatment history alone.

## Core Classification (4 fields)

### 1. topic
**REQUIRED** - Every question must have exactly one topic. Never null.

| Topic | Use When... |
|-------|-------------|
| **Treatment selection** | Choosing WHICH drug/regimen to use for a patient (patient vignette + "which therapy") |
| **Treatment indication** | Whether a patient MEETS CRITERIA for a treatment — eligibility, candidacy, indications. "Which patients are candidates?", "Is this patient eligible?" |
| **Clinical efficacy** | Response rates, survival outcomes, clinical trial results, "key findings" |
| **Safety profile** | Knowing WHAT AEs to expect or recognize — awareness, anticipation. "What are the common AEs?", "Which symptoms should you monitor?" |
| **AE management** | What to DO about an AE already occurring — hold drug, dose-reduce, manage. "Patient develops toxicity, what is the next step?" |
| **Biomarker testing** | Molecular testing, companion diagnostics, test interpretation |
| **Mechanism of action** | Drug targets, pathways, MOA |
| **Diagnosis** | Staging, imaging, diagnostic procedures |
| **Prognosis** | Risk stratification, recurrence scores, survival prediction |
| **Study design** | Trial design, patient enrollment criteria, endpoints |
| **Multidisciplinary care** | Team-based care, referrals, care coordination |
| **Shared decision-making** | Patient-provider communication, discussing treatment options with patients, addressing patient concerns/preferences |
| **Disparities in care** | Racial/ethnic disparities, inequities in access/outcomes |
| **Barriers to care** | Obstacles to treatment (cost, access, adherence) |
| **Surgical management** | Surgical approach and decision-making: BCS vs mastectomy, axillary surgery (SLNB vs ALND), surgical margins, reconstruction (implant vs autologous, prepectoral vs submuscular), anesthesia for breast surgery (PECS blocks), timing of surgery relative to systemic therapy |
| **Radiation therapy management** | Radiation planning, dosing, sequencing: whole-breast RT, post-mastectomy RT (PMRT), hypofractionation, accelerated partial breast irradiation (APBI), regional nodal irradiation, boost techniques, radiation side effects |

### 5-9. treatment_1 through treatment_5
**OPTIONAL** - Only tag fundable drugs of pharmaceutical interest.

**CRITICAL RULES:**
- DO NOT tag: chemotherapy, radiation, surgery, hormonal backbone (fulvestrant, letrozole when combined with fundable drug)
- For combinations with one fundable drug, tag ONLY the fundable drug
- Put each drug in a separate field (treatment_1, treatment_2, etc.)
- Use `null` for unused slots
- **CAPITALIZATION:** Always capitalize the first letter of drug names in treatment fields. Use `"Trastuzumab deruxtecan"` (NOT `"trastuzumab deruxtecan"`), `"Sacituzumab govitecan"` (NOT `"sacituzumab govitecan"`), `"Pembrolizumab"` (NOT `"pembrolizumab"`), etc.

**Fundable drug categories:**
- Targeted therapies (TKIs, mAbs)
- Antibody-drug conjugates (ADCs)
- Immunotherapies (checkpoint inhibitors, CAR-T, bispecifics)
- Novel agents in clinical trials


### treatment_1-5: Critical Tagging Rules

**Rule 1: Subject vs Comparator**
Tag ONLY the drug that is the SUBJECT of the teaching point, not comparators.

EXAMPLE:
Q: "Which AE differentiates Drug A from Drug B and Drug C?"
A: "Specific toxicity"
✓ treatment_1: Drug A (subject being differentiated)
✗ DO NOT TAG: Drug B, Drug C (just comparators)

**Rule 2: Treatment as Clinical Context**
Tag treatments when they provide CONTEXT for interpreting a clinical finding.

**Rule 3: Drug Class → Specific Approved Drugs**
When the correct answer is a drug CLASS, you MUST tag all approved specific drugs.
Use the disease-specific drug table above to map class → individual drugs.

### 10-14. biomarker_1 through biomarker_5

**DO NOT put in biomarker:**
- **HER2 status when already in disease_type** → If disease_type is "HER2+", don't also tag biomarker "HER2" (redundant). Exception: questions specifically about biomarker TESTING
- **Drug targets** → Use `drug_target` instead (e.g., PD-L1 as drug target vs PD-L1 as biomarker for treatment selection)
- **Resistance mutations** → Use `resistance_mechanism` instead

**OPTIONAL** - Tag when clinically relevant AND not redundant with disease_type.

**REDUNDANCY RULE:** If biomarker is already captured in disease_type, use null
- Example: If disease_type is "EGFR-mutated", don't also tag biomarker: "EGFR mutation"

**EXCEPTION:** If topic is "Biomarker testing", you MAY tag the biomarker being tested

**Common values (use these when applicable, but may use other values if confident):**
- Predictive: `"PD-L1"`, `"EGFR mutation"`, `"ALK"`, `"HER2"`, `"BRCA1/2"`, `"MSI-H"`, `"TMB"`, `"PIK3CA mutation"`, `"ESR1 mutation"`, `"FES F18 PET"`
- Prognostic: `"Ki-67"`, `"Oncotype DX"`, `"MammaPrint"`
- Resistance: `"T790M"`, `"ESR1 mutation"`, `"C797S"`

**NORMALIZATION: Always use the full mutation name in biomarker fields:**
- Use `"PIK3CA mutation"` (NOT just `"PIK3CA"`) — the mutation status is what's clinically actionable
- Use `"ESR1 mutation"` (NOT just `"ESR1"`) — ESR1 mutation via ctDNA drives oral SERD selection
- This applies to both the biomarker fields AND the resistance_mechanism field

### 15-19. trial_1 through trial_5
**CONDITIONAL** — Tag behavior depends on topic:

**Always tag** when a trial name is **explicitly mentioned** in the question stem or answer choices (any topic).

**Tag ALL relevant trials, not just the most prominent one.** When a clinical scenario involves multiple trials (e.g., a patient treated per KEYNOTE-522 neoadjuvant regimen who then receives adjuvant olaparib per OlympiA), tag both trials. Use trial_1, trial_2, etc. for each.

**Inference ALLOWED** only when topic is:
- "Clinical efficacy" — Infer from drug + indication + efficacy context
- "Study design" — Infer from drug + indication + study methodology context

**Inference NOT ALLOWED** for all other topics (Treatment selection, Treatment indication, AE management, Biomarker testing, Mechanism of action, Patient selection, Supportive care, Monitoring/follow-up). For these topics, use null unless the trial is explicitly named in the stem or answer choices.

## Group C: Safety/Toxicity (7 fields)

### 45-49. toxicity_type_1 through toxicity_type_5
**OPTIONAL** - Can be null. Tag when specific adverse events are discussed.

**MULTI-VALUE FIELD:** Use separate slots for multiple toxicity types.

**When to tag:**
- AE explicitly mentioned in question, answer, or patient vignette
- Safety profile question asks "what AEs to expect?"
- AE management question describes specific toxicity

**When null is appropriate:**
- No toxicities discussed
- Efficacy-only questions

**Common Breast Cancer Treatment Toxicities:**

**Anthracycline-related (doxorubicin, epirubicin):**
- `"Cardiomyopathy"` - Cumulative dose-dependent, IRREVERSIBLE (lifetime limit ~450-550 mg/m² doxorubicin)
- `"Neutropenia"` - Dose-limiting, especially with dose-dense regimens
- `"Febrile neutropenia"` - Risk with dose-dense AC, G-CSF prophylaxis
- `"Nausea/vomiting"` - Highly emetogenic
- `"Alopecia"` - Very common (near universal)
- `"Mucositis/stomatitis"` - Oral/GI mucosal inflammation

**Taxane-related (paclitaxel, docetaxel, nab-paclitaxel):**
- `"Peripheral neuropathy"` - Cumulative, dose-limiting (paclitaxel > docetaxel)
- `"Neutropenia"` - Common with docetaxel; less with weekly paclitaxel
- `"Myalgia/arthralgia"` - Muscle and joint pain (common with paclitaxel)
- `"Fluid retention"` - Docetaxel-specific (edema, pleural effusion, weight gain)
- `"Nail changes"` - Docetaxel-specific (onycholysis, discoloration)
- `"Alopecia"` - Very common
- `"Hypersensitivity reaction"` - Paclitaxel (Cremophor-based), requires premedication

**HER2-targeted therapy-related (trastuzumab, pertuzumab, T-DM1, margetuximab):**
- `"Cardiomyopathy"` - Trastuzumab-related, REVERSIBLE (unlike anthracycline), LVEF monitoring required
- `"Diarrhea"` - Pertuzumab (most common AE, especially with trastuzumab + docetaxel)
- `"Infusion reactions"` - First infusion risk with trastuzumab
- `"Thrombocytopenia"` - T-DM1 specific (dose-limiting)
- `"Hepatotoxicity"` - T-DM1 specific (transaminase elevation, nodular regenerative hyperplasia)
- `"Peripheral neuropathy"` - T-DM1 specific

**ADC-related (T-DXd, sacituzumab govitecan, other Trop-2/HER2 ADCs):**
- `"Interstitial lung disease (ILD)"` - T-DXd CRITICAL toxicity (potentially fatal, any grade requires attention)
- `"Neutropenia"` - Common with sacituzumab govitecan (dose-limiting)
- `"Diarrhea"` - Sacituzumab govitecan (early-onset, irinotecan metabolite SN-38)
- `"Nausea/vomiting"` - Common with both T-DXd and sacituzumab govitecan
- `"Alopecia"` - Common with ADCs
- `"Fatigue"` - Common systemic toxicity

**CDK4/6 inhibitor-related (palbociclib, ribociclib, abemaciclib):**
- `"Neutropenia"` - Palbociclib/ribociclib (most common, dose-limiting, usually manageable)
- `"Diarrhea"` - Abemaciclib-specific (most common AE, dose-limiting)
- `"QT prolongation"` - Ribociclib-specific (requires ECG monitoring)
- `"Hepatotoxicity"` - All CDK4/6i (transaminase elevation, monitor LFTs)
- `"VTE"` / `"Pulmonary embolism"` - All CDK4/6i (increased thromboembolic risk)
- `"Fatigue"` - Common across class

**Endocrine therapy-related (tamoxifen, aromatase inhibitors, fulvestrant):**
- `"Hot flashes"` - Very common with all endocrine therapies
- `"Arthralgias"` - Aromatase inhibitor-specific (joint pain/stiffness, common reason for discontinuation)
- `"Osteoporosis"` / `"Bone loss"` - Aromatase inhibitor-specific (requires bone density monitoring)
- `"Endometrial cancer risk"` - Tamoxifen-specific (increased risk, requires monitoring)
- `"VTE"` / `"Thromboembolism"` - Tamoxifen-specific (DVT, PE risk)
- `"Vaginal dryness"` / `"Sexual dysfunction"` - Common with all endocrine therapies
- `"Mood changes"` / `"Depression"` - Common with endocrine therapies

**Immunotherapy-related (pembrolizumab in TNBC):**
- `"Immune-related pneumonitis"` - Lung toxicity requiring steroids
- `"Immune-related colitis"` - GI toxicity
- `"Immune-related hepatitis"` - Liver toxicity
- `"Immune-related thyroiditis"` - Thyroid dysfunction (hypothyroidism, hyperthyroidism)
- `"Immune-related myocarditis"` - Rare but serious cardiac toxicity
- `"Adrenal insufficiency"` - Immune-related endocrine toxicity
- `"Rash"` - Immune-related dermatologic toxicity

**PARP inhibitor-related (olaparib, talazoparib):**
- `"Anemia"` - Most common hematologic toxicity (talazoparib > olaparib)
- `"Nausea"` - Common GI toxicity
- `"Fatigue"` - Common
- `"Thrombocytopenia"` - Talazoparib (more myelosuppressive than olaparib)
- `"MDS/AML"` - Rare but serious long-term risk

**PI3K/AKT inhibitor-related (alpelisib, capivasertib):**
- `"Hyperglycemia"` - Alpelisib CRITICAL toxicity (on-target, PI3K pathway), requires monitoring
- `"Diarrhea"` - Common with both alpelisib and capivasertib
- `"Rash"` - Alpelisib-specific (maculopapular, can be severe)
- `"Stomatitis"` - Common oral toxicity

**Capecitabine-related:**
- `"Hand-foot syndrome"` - Palmar-plantar erythrodysesthesia (dose-limiting, characteristic AE)
- `"Diarrhea"` - GI toxicity
- `"Nausea/vomiting"` - GI toxicity

**Radiation therapy-related:**
- `"Radiation dermatitis"` - Skin erythema, desquamation
- `"Radiation pneumonitis"` - Lung toxicity (especially left-sided, concurrent with systemic therapy)
- `"Lymphedema"` - Post-axillary radiation (especially with axillary dissection)
- `"Cardiac toxicity"` - Left-sided radiation (long-term coronary artery disease risk)
- `"Rib fracture"` - Post-radiation bone fragility

**Examples:**
- "LVEF decline on trastuzumab" → toxicity_type_1: "Cardiomyopathy"
- "ILD on T-DXd" → toxicity_type_1: "Interstitial lung disease (ILD)"
- "Grade 3 diarrhea on abemaciclib" → toxicity_type_1: "Diarrhea"
- "Hyperglycemia requiring insulin on alpelisib" → toxicity_type_1: "Hyperglycemia"

**OPTIONAL** - Can be null. Tag when organ system affected by toxicity is relevant.

- `"Neurologic"` - Peripheral neuropathy (taxanes, T-DM1)

## Group F: Question Format/Quality (13 fields)

This section evaluates both the educational level AND the structural quality of the question for internal QA.

### 54. cme_outcome_level
**OPTIONAL** - Moore's level of CME outcome.

**Valid values:**
- `"3 - Knowledge"` - Question tests recall/recognition of information
- `"4 - Competence"` - Question tests application of knowledge to a clinical scenario

**CRITICAL RULE:** If the clinical vignette is NOT REQUIRED to answer the question, it's **Level 3 (Knowledge)**, not Level 4.

**Test:** Could someone answer this question correctly WITHOUT reading the patient details?
- YES → `"3 - Knowledge"` (the vignette is just context/decoration)
- NO → `"4 - Competence"` (must apply knowledge to the specific patient)

**How to determine:**
- **Level 3 (Knowledge):** "What is the mechanism of action of X?", "Which biomarker predicts response to Y?"
- **Level 4 (Competence):** Patient vignette asking "What is the best treatment for this patient?", case-based decision making

### 55. data_response_type
**OPTIONAL** - Type of response the question elicits (content type).

**Valid values:**
- `"Numeric"` - Answer is a specific number (e.g., "What was the ORR?")
- `"Qualitative"` - Answer is descriptive (e.g., "Which best describes the mechanism?")
- `"Comparative"` - Answer compares options (e.g., "Which is preferred?")
- `"Boolean"` - Answer is Yes/No (e.g., "Is X approved for Y?")

### 56. stem_type
**OPTIONAL** - Format of the question stem.

**Valid values:**
- `"Clinical vignette"` - Patient case scenario (tests application)
- `"Direct question"` - Straightforward question without patient context (tests recall)
- `"Incomplete statement"` - "The mechanism of X is..." format

### 57. lead_in_type
**OPTIONAL** - Type of question lead-in.

**Valid values:**
- `"Standard"` - "Which of the following...", "What is the most..."
- `"Negative (EXCEPT/NOT)"` - "All of the following EXCEPT...", "Which is NOT..."
- `"Best answer"` - "What is the BEST...", "most appropriate", "most likely"
- `"True statement"` - "Which statement is TRUE...", "Which is correct..."

### 58. answer_format
**OPTIONAL** - Structure of answer options.

**Valid values:**
- `"Single best"` - Standard single correct answer format
- `"Compound (A+B)"` - "A and B", "Both A and C" as options
- `"All of above"` - "All of the above" is an answer option
- `"None of above"` - "None of the above" is an answer option
- `"True-False"` - True/False format

### 59. answer_length_pattern
**OPTIONAL** - Relative length of answer options. **DEFAULT: "Uniform"**

**CRITICAL: Most questions are Uniform.** Only use "Variable" when there is a dramatically obvious length difference (one option is a brief phrase while another is a full sentence).

**Valid values:**
- `"Uniform"` - **DEFAULT.** All options are roughly similar length. This includes:
  - All short phrases (2-6 words each)
  - All medium phrases (5-10 words each)
  - All sentences (10-20 words each)
  - Minor variation within the same general category
- `"Variable"` - **RARE.** Only when one option is dramatically longer (3x+). Example: one option is 3 words, another is 15+ words.
- `"Correct longest"` - Correct answer is dramatically longer than all distractors (flaw indicator)
- `"Correct shortest"` - Correct answer is dramatically shorter than all distractors (flaw indicator)

**DEFAULT: If `distractor_homogeneity` = "Homogeneous", ALWAYS tag `answer_length_pattern` = "Uniform".**

Override to "Variable" or "Correct longest/shortest" ONLY when the length difference is EGREGIOUS — e.g., one answer is a single word and another is a full sentence with additional clinical detail.

Do NOT tag "Variable" just because:
- One option has a few extra words
- Compound answers are longer ("Both A and B" vs "A only")
- Drug name lengths differ

**When in doubt with Homogeneous distractors → tag "Uniform".**

**Examples:**
- **Uniform**: "Myeloma related to Gaucher disease" / "Progressive Gaucher disease" / "Gaucher disease-related osteoporosis" (all 3-5 words) → Uniform
- **Uniform**: Drug names, regimen names, or any set of options that are all in the same general length range → Uniform
- **Variable**: One option is "Pembrolizumab" (1 word) while another is "Combination of platinum-based chemotherapy with maintenance pemetrexed following disease stabilization" (10+ words) → Variable

**Note:** "Correct longest" and "Correct shortest" are testwise cues indicating potential item writing issues - only use when the correct answer stands out dramatically in length.

### 60. distractor_homogeneity
**OPTIONAL** - Whether answer options are structurally interchangeable or substantively different.

**Valid values:**

**"Homogeneous"** — All options are the **same TYPE and similar SPECIFICITY** (interchangeable format, just swapping one entity for another):
- All specific drug names: "Osimertinib" / "Erlotinib" / "Afatinib" / "Gefitinib"
- All regimen names: "Dara-VRd" / "VRd" / "KRd" / "Isa-VRd"
- All biomarkers: "EGFR" / "ALK" / "ROS1" / "KRAS"
- All short clinical phrases of similar scope

**"Heterogeneous"** — Options differ in TYPE, SPECIFICITY, or are **substantively different statements**:
- Mixed categories (drug name vs. dosing schedule vs. monitoring plan)
- Different scope/specificity (brief phrase vs. complex multi-clause statement)
- Substantively different clinical assertions even if sharing a broad theme
- Example: "This patient's age does not exclude transplant" / "Deeper remission will not be possible" / "If deferred, transplant no longer an option" / "Transplant unlikely to prolong TTP" → **Heterogeneous** (different clinical rationales requiring independent evaluation)

**Cross-modality treatment options:** When ALL answer options are viable treatment approaches for the same clinical problem — even if they mix modalities (drug therapy, radiation, surgery) — classify as **"Homogeneous"**. They are all "treatments" and interchangeable in category.
- Example: A: "SRS followed by tucatinib + trastuzumab + capecitabine", B: "SRS followed by trastuzumab + capecitabine", C: "Surgery followed by WBRT", D: "WBRT" → **Homogeneous** (all are viable treatment approaches for brain metastases)

**Decision rule:** If options are interchangeable in format (just swapping one entity for another), use "Homogeneous". If each option must be evaluated on its own clinical merits, use "Heterogeneous".

### Item Writing Flaws (61-66)

These 6 fields are **boolean** (true/false). Set to `true` if the flaw is present, `false` if not.

### 61. flaw_absolute_terms
**BOOLEAN** - Answer options contain absolute terms.

Set `true` if any answer option contains: "always", "never", "all", "none", "only", "must", "cannot"

**Why it matters:** Test-wise examinees know absolutes are usually wrong.

### 62. flaw_grammatical_cue
**BOOLEAN** - Stem grammar reveals the answer.

Set `true` if:
- Stem ends with "a" or "an" and only one option starts with correct vowel/consonant
- Stem uses singular but some options are plural (or vice versa)
- Subject-verb agreement issues that eliminate options

**Why it matters:** Grammar matching guides to correct answer without content knowledge.

### 63. flaw_implausible_distractor
**BOOLEAN** - Contains obviously wrong options.

Set `true` if any distractor is clearly wrong to anyone with basic clinical knowledge (e.g., "Homeopathy" as treatment option for cancer, a drug from completely wrong class).

**Why it matters:** Reduces effective choices and inflates guessing success.

### 64. flaw_clang_association
**BOOLEAN** - Answer shares unusual words/phrases with stem.

Set `true` if the correct answer contains a distinctive word or phrase that also appears in the stem (word repetition beyond common clinical terms).

**Why it matters:** Word repetition guides to correct answer without actual knowledge.

### 65. flaw_convergence_vulnerability
**BOOLEAN** - Correct answer combines elements from multiple options.

Set `true` if:
- Multiple options share common elements
- The correct answer has the most overlapping elements with other options
- Test-taker could reason to correct answer by finding the "most complete" option

**Why it matters:** Allows reasoning to answer without content knowledge.

### 66. flaw_double_negative
**BOOLEAN** - Negative stem combined with negative option.

Set `true` if:
- Stem contains "NOT", "EXCEPT", "LEAST likely", etc. AND
- One or more answer options contain negative phrasing

**Why it matters:** Increases cognitive load, tests reading comprehension rather than knowledge.

# Universal Tagging Rules

These rules apply to ALL diseases. Disease-specific rules in the disease module take precedence when there is a conflict.

---

## Core Principles

### 1. Use Null Appropriately
- Use `null` when a value cannot be determined with confidence
- Use `null` for unused multi-value slots (e.g., treatment_3 through treatment_5 when only 2 drugs)
- **Never guess** - if uncertain, use `null`

### 2. Consider the Correct Answer
- The correct answer often reveals context not in the question
- Subtype, treatment line, and stage may be implied by the answer
- Use answer content to inform your tagging

### 3. Topic Definitions (Apply Consistently)

| Topic | Key Trigger | Example |
|-------|-------------|---------|
| **Treatment selection** | Choosing WHICH drug/regimen (patient vignette + "which therapy") | "A 55-year-old woman... which regimen is most appropriate?" |
| **Treatment indication** | Whether patient MEETS CRITERIA — eligibility, candidacy | "Which patients are candidates for X?", "Is this patient eligible?" |
| **Clinical efficacy** | Trial results, response rates, survival | "What was the ORR in TRIAL-X?" |
| **Safety profile** | Knowing WHAT AEs to expect/recognize (awareness) | "What are the common AEs?", "Which symptoms should you monitor?" |
| **AE management** | What to DO about an AE already occurring | "Patient develops CRS, what is the next step?" |
| **Biomarker testing** | Testing, companion diagnostics | "Which patients should be tested for X?" |
| **Mechanism of action** | How drug works | "What is the target of X?" |
| **Study design** | Enrollment, endpoints, trial structure | "Which patients were enrolled in TRIAL-X?" |

### 4. Tag from CORRECT answer only
Only tag treatments, biomarkers, toxicities, and other clinical entities from the CORRECT answer. Do NOT tag entities from incorrect options (distractors).
- **Standard questions ("Which of the following...")**: Tag ONLY entities from the correct answer
- **Negative lead-ins ("All EXCEPT", "Which is NOT")**: Tag the INCORRECT options (distractors) instead, as they represent the true/factual statements. The correct answer is the exception and should NOT be tagged as a treatment/biomarker.
- **"Which statement is FALSE/INCORRECT?"**: The correct answer is the false statement — do NOT treat it as the recommended approach. Tag clinical entities from the TRUE distractors instead.
- **Compound answers ("A and B", "All of the above")**: Tag all components that make up the correct answer
- **Example (standard)**: If Q asks "Which toxicity is most associated with X?" and A: CRS, B: ICANS, C: Neuropathy, D: Hepatotoxicity with correct answer A, ONLY tag CRS as toxicity, NOT the distractors.
- **Example (EXCEPT — toxicity)**: Q asks "Which AE is NOT seen with sacituzumab govitecan or datopotamab deruxtecan?" with correct answer D. Tag toxicities from options A, B, C (the real AEs), NOT option D (which is the AE that does NOT occur).
- **Example (FALSE — treatment)**: Q asks about DCIS: "Which statement is FALSE?" and correct answer mentions mastectomy. Do NOT tag mastectomy as the treatment of choice — it is the false statement. Tag clinical entities from the true distractors instead.

## Treatment Tagging Rules

### Fundable Drugs Only
Tag ONLY drugs of pharmaceutical interest for CME funding:
- Targeted therapies (TKIs, mAbs)
- ADCs
- Immunotherapies (checkpoint inhibitors, CAR-T, bispecifics)
- Novel agents

### DO NOT Tag
- **Chemotherapy agents**: carboplatin, paclitaxel, docetaxel, capecitabine, doxorubicin, etc.
- **Hormonal backbone**: fulvestrant, letrozole, anastrozole, tamoxifen (when combined with fundable drug)
- **Radiation therapy**
- **Surgical procedures**
- **Supportive care**: G-CSF, antiemetics, steroids (unless explicitly the focus)

### Combination Therapy
- For combinations with ONE fundable drug: tag only the fundable drug
  - "ribociclib + letrozole" → treatment_1: "ribociclib" (letrozole not tagged)
  - "pembrolizumab + chemotherapy" → treatment_1: "pembrolizumab"
- For combinations with MULTIPLE fundable drugs: tag each in separate slots
  - "trastuzumab + pertuzumab" → treatment_1: "trastuzumab", treatment_2: "pertuzumab"

### Drug Class Expansion
When question mentions a drug class without specific agent:
- Expand to all approved drugs in that class (as of question STARTDATE)
- Put each drug in a separate treatment slot
- Example: "CDK4/6 inhibitor" for metastatic → treatment_1-3: palbociclib, ribociclib, abemaciclib

**IMPORTANT: Drug class expansion applies to AE management and Safety profile topics too.**
When a question asks about managing an AE or safety profile of a drug CLASS (not a specific drug), infer and tag the specific approved drugs using disease context and STARTDATE:
- Example: "Patient on a CDK4/6 inhibitor develops neutropenia" → Infer all approved CDK4/6 inhibitors for this setting
- Example: "ADC-associated ILD in breast cancer" → Infer approved ADCs (trastuzumab deruxtecan, sacituzumab govitecan, etc.)
- The causative drug class is clinically relevant — funders need to know WHICH specific drugs are associated with the AE

## Biomarker Redundancy Rules

### Core Rule
**DO NOT tag a biomarker if it is already captured in disease_type**

| If disease_type is... | Don't tag biomarker: |
|-----------------------|---------------------|
| `"EGFR-mutated"` | `"EGFR mutation"` |
| `"HER2+"` | `"HER2"` |
| `"ALK-positive"` | `"ALK"` |
| `"MSI-H/dMMR"` | `"MSI-H"` |
| `"BRCA-mutated"` | `"BRCA1/2"` |

### Exception: Biomarker Testing Topic
If topic is `"Biomarker testing"` and question is about testing for a specific biomarker, you MAY tag that biomarker even if it's in disease_type.

### Non-Redundant Biomarkers
Always OK to tag biomarkers that are ADDITIONAL to disease_type:
- PIK3CA (in HR+/HER2- breast cancer → tag as biomarker)
- ESR1 (in HR+/HER2- → tag as biomarker)
- PD-L1 (in any disease → tag as biomarker)
- TMB (in any disease → tag as biomarker)

## Trial Inference Rules

### When to Infer Trial Names
**ALLOWED** for topics: `"Clinical efficacy"`, `"Study design"`
- Use drug + indication + population to confidently identify the trial
- Must have high confidence - if uncertain, use `null`

**NOT ALLOWED** for all other topics: `"Treatment selection"`, `"Treatment indication"`, `"AE management"`, `"Biomarker testing"`, `"Mechanism of action"`, `"Patient selection"`, `"Supportive care"`, `"Monitoring/follow-up"`, `"Safety profile"`
- These ask "which drug to use" or about non-efficacy concepts — not "what did the trial show"
- Only tag trial if explicitly named in the question stem or answer choices

### Inference Requirements
All of these must be clear to infer a trial:
1. Specific drug(s)
2. Specific indication/disease state
3. Specific treatment line or setting
4. Clinical efficacy or study design context

## Group A-F Field Guidelines

### Group A: Treatment Metadata
- **drug_class**: Use when discussing a class of drugs (e.g., "EGFR TKI")
- **drug_target**: Use when discussing molecular target
- **prior_therapy**: Only tag when explicitly mentioned as prior treatment
- **resistance_mechanism**: Only tag when explicitly discussed

- **metastatic_site**: Only tag sites explicitly mentioned

- **special_population**: Tag patient characteristics that affect treatment (elderly, organ dysfunction)

### Group C: Safety/Toxicity
- **toxicity_type**: Tag specific toxicities discussed
- **toxicity_organ**: Tag organ system if toxicity focus
- **toxicity_grade**: Tag grade if mentioned (Grade 3, Grade >=3)

### Group D: Efficacy/Outcomes
- **efficacy_endpoint**: Tag ONLY for "Clinical efficacy" topic OR when endpoint explicitly named in stem
- **outcome_context**: Tag analysis type (primary endpoint, subgroup, interim)
- **clinical_benefit**: Tag nature of benefit (statistically significant, clinically meaningful)

### Group E: Evidence/Guidelines
- **guideline_source**: Tag ONLY when guideline body is explicitly named in stem/answers (NCCN, ASCO, ESMO)
- **evidence_type**: Tag study design (Phase 3 RCT, real-world evidence)
- **trial_phase**: Tag phase if mentioned

### Group F: Question Format
- **cme_outcome_level**:
  - Level 3 (Knowledge): Tests recall/recognition
  - Level 4 (Competence): Tests application to clinical scenario
- **data_response_type**:
  - Numeric: Answer is a number
  - Qualitative: Answer is descriptive
  - Comparative: Answer compares options
- **endpoint_type**: Only for questions about trial endpoints

## Special Situations

### AE as Secondary Condition
When question asks about managing an AE (e.g., MDS from therapy):
- Keep the PRIMARY disease being treated as the disease context
- Use topic = "AE management" or "Safety profile"
- DO NOT change disease_state to the AE/complication

### Pan-Tumor Questions
For questions not specific to one disease:
- disease_type: `null`
- disease_stage: `null` (unless clearly stated)
- Focus on the treatment/biomarker being discussed

### Unknown Trial
If a trial is mentioned but you don't recognize it:
- Still tag the trial name as stated
- Tag what you can infer from context
- Use `null` for fields you cannot determine
