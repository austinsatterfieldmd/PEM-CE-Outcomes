### 2. disease_stage
**OPTIONAL** - Can be null, but try hard to infer before accepting null.

**Valid values:**
- `"Early-stage"` - Stage I, II, or III (non-metastatic, curative intent)
- `"Metastatic"` - Stage IV or metastatic disease

**When to infer (TRY THESE FIRST):**
- Trial context: monarchE → Early-stage, DESTINY-Breast04 → Metastatic
- Treatment line: adjuvant/neoadjuvant → Early-stage, 1L/2L+ → Metastatic
- Drug indication: Most targeted therapies imply metastatic unless adjuvant context
- Abbreviations: "mBC", "MBC" → Metastatic

**When null is appropriate:**
- General biomarker testing questions without disease context
- MOA questions about a drug without indication specified
- Truly pan-stage questions (e.g., "What biomarkers should be tested in all breast cancer?")

**Triggers for Metastatic:** "mBC", "MBC", "metastatic", "stage IV", "advanced breast cancer"
**Triggers for Early-stage:** "stage I", "stage II", "stage III", "early-stage", "resectable", "operable", "localized", "non-metastatic", "adjuvant", "neoadjuvant"

**IMPORTANT:** Stage III is "Early-stage" (non-metastatic)

### 3. disease_type
**OPTIONAL but CRITICAL** - Try very hard to determine. Subtyping is the #1 priority for breast cancer.

**Valid values (receptor-based subtypes):**
- `"HR+/HER2-"` - ER+ and/or PR+, HER2-negative (IHC 0 without membrane staining)
- `"HR+/HER2+"` - ER+ and/or PR+, HER2 IHC 3+ or 2+/ISH+
- `"HER2+"` - HER2 IHC 3+ or 2+/ISH+ (HR-negative or HR status not specified)
- `"HER2-low"` - HER2 IHC 1+ or 2+/ISH- (distinct from HER2+)
- `"HER2-ultralow"` - HER2 IHC 0 with membrane staining (actionable per DESTINY-Breast06)
- `"Triple-negative"` or `"TNBC"` - ER-, PR-, HER2-

**Valid values (histologic subtypes):**
- `"DCIS"` - Ductal carcinoma in situ (non-invasive, stage 0)
- `"ILC"` - Invasive lobular carcinoma (distinct biology, ~10-15% of breast cancers)

**When to infer (TRY THESE FIRST):**
- Trial context: DESTINY-Breast04 → HER2-low, monarchE → HR+/HER2-, KEYNOTE-522 → Triple-negative
- Drug: CDK4/6 inhibitors → HR+/HER2-, sacituzumab govitecan → TNBC or HR+/HER2-, T-DXd → HER2+, HER2-low, or HER2-ultralow
- Correct answer often reveals subtype even when question doesn't state it
- Histology keywords: "ductal carcinoma in situ" → DCIS, "lobular" → ILC

**When null is appropriate:**
- General breast cancer questions without subtype context (e.g., "What is the role of surgery?")
- Questions about pan-subtype topics (e.g., general bone metastasis management)
- Truly indeterminate after exhausting inference sources

**Subtyping Hierarchy:**
1. **Histologic subtype takes precedence when question focuses on histology**: If question is specifically about DCIS or ILC management, use `"DCIS"` or `"ILC"`
2. **For invasive cancer (non-DCIS)**: Determine HER2 status: HER2+ > HER2-low > HER2-ultralow > HER2-negative
3. If HER2+: Check HR status → If HR+ explicit, use `"HR+/HER2+"`; otherwise use `"HER2+"`
4. If HER2-negative, determine HR: HR+ → "HR+/HER2-", HR- → "Triple-negative"

**Edge cases:**
- **HR+/HER2+**: Use `"HR+/HER2+"` when both HR and HER2 positivity are explicit
- **HR+/HER2-low or HR+/HER2-ultralow (critical classification logic):**
  - If topic is "Treatment selection" and answer involves **T-DXd** → Use `"HER2-low"` or `"HER2-ultralow"` (T-DXd is the actionable driver)
  - Otherwise → Use `"HR+/HER2-"` (CDK4/6i, endocrine therapy drive treatment)
  - *Rationale:* T-DXd is the only approved drug specifically for HER2-low/-ultralow. If the question is about T-DXd, the HER2 status is clinically actionable and should be tagged.
- If question discusses ILC with receptor status (e.g., "HR+ ILC"), use `"ILC"` if histology is the focus, otherwise use receptor subtype
- DCIS is always "Early-stage" (stage 0) for disease_stage

## Important Guidelines

1. **Subtype is CRITICAL** - Try very hard to determine HR/HER2 status
2. **HER2 precedence** - If HER2+, use that as disease_type regardless of HR status
3. **No redundant biomarkers** - Don't tag HER2 as biomarker if it's in disease_type (unless biomarker testing question)
4. **HER2-low is distinct** - HER2-low ≠ HER2+ (different treatment eligibility)
5. **Stage III is early-stage** - Stage III gets neoadjuvant/adjuvant therapy, not metastatic treatment
6. **Consider the answer** - Subtype may be revealed in the correct answer
7. **Tag content from the CORRECT answer only**: Only tag treatments, biomarkers, and other entities that appear in the CORRECT answer, NOT the distractors (incorrect options). The distractors are wrong/not applicable by definition.
   - **Standard questions ("Which of the following...")**: Tag ONLY entities from the correct answer
   - **Negative lead-ins ("All EXCEPT", "Which is NOT")**: Tag the INCORRECT options instead, as they represent the true/factual statements
   - **Compound answers ("A and B", "All of the above")**: Tag all components that make up the correct answer
8. **Fundable drugs only** - Don't tag chemotherapy or hormonal backbone therapies
9. **Infer from context** - Trials, drugs, and clinical context can imply stage, type, line, and biomarker

### Special Cases

9. **BIA-ALCL (Breast Implant-Associated Anaplastic Large Cell Lymphoma)**:
   - BIA-ALCL is classified under "Breast cancer" for Stage 1 (disease_state)
   - For Stage 2 tagging: topic = "AE management" (it's a rare complication of breast reconstruction)
   - disease_type = null (not a receptor subtype)
   - BIA-ALCL occurs in patients who had breast implants, often for reconstruction after breast cancer treatment
