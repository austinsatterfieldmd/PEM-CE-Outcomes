## Group A: Core Fields (20 fields)

### 1. topic
**REQUIRED** - Every question must have exactly one topic. Never null.

| Topic | Use When... |
|-------|-------------|
| **Treatment selection** | Choosing WHICH drug/regimen for a patient |
| **Treatment indication** | Whether a patient MEETS CRITERIA for treatment |
| **Clinical efficacy** | Response rates, survival outcomes, trial results |
| **Safety profile** | Knowing WHAT AEs to expect |
| **AE management** | What to DO about an AE already occurring |
| **Prophylaxis** | How to PREVENT toxicities BEFORE they occur |
| **Biomarker testing** | CD20, CD30, molecular testing |
| **Mechanism of action** | Drug targets, pathways |
| **Diagnosis** | Lymphoma subtype determination |
| **Prognosis** | Risk factors, outcomes prediction |

### ⚠️ Topic Disambiguation

**"Indication" vs "Efficacy":**
- "Who should receive CAR-T therapy?" → **Treatment indication**
- "What is the CR rate with axi-cel?" → **Clinical efficacy**

**"Safety profile" vs "Prophylaxis" vs "AE management":**
- **Safety profile** = knowing WHAT AEs to expect (awareness)
- **Prophylaxis** = how to PREVENT toxicities BEFORE they occur
- **AE management** = what to DO after an AE has occurred

- The causative drug class is clinically relevant — funders need to know WHICH specific drugs are associated with the AE

## Group G: Question Format/Quality (15 fields)

### 56. cme_outcome_level
**Values:** `"3 - Knowledge"` or `"4 - Competence"`

**CRITICAL RULE:** If the clinical vignette is NOT REQUIRED to answer the question, it's **Level 3 (Knowledge)**, not Level 4.

**Test:** Could someone answer correctly WITHOUT reading the patient details?
- YES → `"3 - Knowledge"`
- NO → `"4 - Competence"`

### 57-60. Format fields
- `data_response_type`: "Numeric", "Qualitative", "Comparative", "Boolean"
- `stem_type`: "Clinical vignette", "Direct question", "Incomplete statement"
- `lead_in_type`: "Standard", "Negative (EXCEPT/NOT)", "Best answer"
- `answer_format`: "Single best", "Compound (A+B)", "All of the above", "None of the above"

### 61. answer_length_pattern
**Values:** "Uniform", "Variable", "Correct longest", "Correct shortest"

**CRITICAL RULE:** If `distractor_homogeneity` = "Homogeneous", then `answer_length_pattern` is almost always "Uniform".

### 62. distractor_homogeneity
**"Homogeneous"** = All options are the SAME TYPE (all drugs, all actions)
**"Heterogeneous"** = Options are DIFFERENT TYPES

### 63-68. Flaw fields (all Boolean: true/false)

- `flaw_grammatical_cue`: true/false

### 69-70. Computed fields
