## CRITICAL: GVHD Context Determination

**GVHD has THREE distinct clinical contexts that MUST be distinguished:**

1. **GVHD Prophylaxis** - Prevention given at time of/after transplant to PREVENT GVHD
2. **GVHD Treatment** - Treatment given AFTER GVHD develops
3. **Acute vs Chronic GVHD** - Different disease types with different treatments

**How to distinguish:**
- "prophylaxis", "prevention", "to prevent GVHD" -> **Prophylaxis context**
- "patient with GVHD", "developed GVHD", "steroid-refractory", "after steroids" -> **Treatment context**
- "day 28 post-transplant", "within 100 days" -> **Acute GVHD**
- "7 months after transplant", "fasciitis", "sclerosis", "chronic" -> **Chronic GVHD**

## CRITICAL: Compound Answer Handling

When the correct answer references options by letter/number (e.g., "B or C", "A and C", "Options 1 and 3"):

1. **DO NOT assume option order matches the original question** - The options you receive may be in a different order than the actual test question.

2. **Deduce the correct answer from clinical reasoning** - Read the substance of each option and determine which ones make sense as the correct answer based on the clinical scenario.

3. **Tag based on the deduced correct content** - Once you identify what the compound answer actually refers to, tag those treatments/concepts.

**Example (GVHD):**
- **Stem:** 55-year-old with steroid-refractory chronic GVHD after prior ibrutinib. Which is the most appropriate next therapy?
- **Correct answer:** "A or B"
- **Options presented:** Belumosudil | Axatilimab | Rituximab | Best supportive care

**How to deduce:** Patient has steroid-refractory chronic GVHD, failed ibrutinib. Both belumosudil and axatilimab are approved for chronic GVHD after prior therapy.

### answer_length_pattern Criteria (CRITICAL - Common Error)

**DEFAULT: If `distractor_homogeneity` = "Homogeneous", ALWAYS tag `answer_length_pattern` = "Uniform".**

Override to "Variable" or "Correct longest/shortest" ONLY when the length difference is EGREGIOUS — e.g., one answer is a single word and another is a full sentence with additional clinical detail.

Do NOT tag "Variable" just because:
- One option has a few extra words
- Compound answers are longer ("Both A and B" vs "A only")
- Drug name lengths differ

**When in doubt with Homogeneous distractors → tag "Uniform".**

Tag "Variable" with Homogeneous distractors ONLY when one answer contains **additional clinical/educational content** the others lack - NOT when length differs due to:
- Grammatical phrasing
- Compound answers ("Both A and B")
- Minor word count differences

**"Uniform"** = All options are CONCEPTUALLY similar in complexity:
- All single drug names: "Ruxolitinib" / "Belumosudil" / "Ibrutinib" -> **Uniform**

**"Variable"** = Options differ in TYPE or COMPLEXITY AND contain different amounts of clinical detail

**"Correct longest/shortest"** = The CORRECT answer is noticeably longer/shorter than ALL distractors

### distractor_homogeneity Criteria (CRITICAL - Common Error)

**"Homogeneous"** = All distractors are the SAME TYPE of answer:
- All drug names: "Ruxolitinib" / "Belumosudil" / "Ibrutinib" / "Axatilimab" -> **Homogeneous**
- All clinical actions: "Increase dose" / "Decrease dose" / "Hold therapy" / "Discontinue" -> **Homogeneous**

**"Heterogeneous"** = Distractors are DIFFERENT TYPES of answers:
- Treatment + Mechanism + Endpoint: "Belumosudil" / "ROCK2 inhibition" / "Skin response" -> **Heterogeneous**

## Important Guidelines for GVHD

1. **Prophylaxis vs Treatment is CRITICAL** - treatment_line: "Prophylaxis" vs "1L", "2L", etc.
2. **Acute vs Chronic in disease_type_1** - Different disease biology, different approved agents
3. **Steroid-refractory in disease_type_2** - Captures disease status (treatment-naive vs refractory)
4. **Numeric line in treatment_line** - 1L (steroids), 2L (first targeted), 3L, 4L+
5. **Organ involvement in symptom fields** - Fasciitis, diarrhea, elevated LFTs, etc.
6. **NIH severity in disease_specific_factor** - NIH mild, moderate, severe
7. **Prior therapy in prior_therapy field** - "After ibrutinib" is prior_therapy
8. **Tag from CORRECT answer only** - Don't tag distractors

## Key Rules for Output

1. **All 70 fields must be present** - Never omit a field; use `null` for inapplicable fields
2. **Exact canonical values** - Use values exactly as specified
3. **Valid JSON** - Proper formatting
4. **One topic only** - Topic is required; never null
5. **Boolean flaw fields** - Use `true` or `false` (not strings, not null)
6. **disease_type_1** - Acute GVHD vs Chronic GVHD (type)
7. **disease_type_2** - Treatment-naive vs Steroid-refractory (status)
8. **treatment_line** - Numeric: Prophylaxis, 1L, 2L, 3L, 4L+ (actual line count)
9. **Organ involvement in symptom fields** - Not disease_specific_factor
10. **Tag from correct answer** - Not distractors
