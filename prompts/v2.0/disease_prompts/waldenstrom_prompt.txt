## WM Overview

**Definition:** Lymphoplasmacytic lymphoma (LPL) with bone marrow involvement and IgM monoclonal gammopathy

**Key features:**
- MYD88 L265P mutation (>90% of cases)
- CXCR4 mutations (30-40%, affect treatment response)
- Hyperviscosity syndrome from IgM
- No staging system - treatment based on symptoms

---

## WM Drug Landscape

### BTK Inhibitors (Mainstay)
| Drug | Notes | Trial |
|------|-------|-------|
| **Ibrutinib** | First approved, MYD88 mutated better | iNNOVATE |
| **Zanubrutinib** | Better tolerated, fewer cardiac AEs | ASPEN |
| **Acalabrutinib** | Used but less WM-specific data | - |

### Other Agents
| Drug | Class | Context |
|------|-------|---------|
| **Rituximab** | Anti-CD20 | Often combined, watch for IgM flare |
| **Bendamustine** | Alkylator | With rituximab (BR) |
| **Bortezomib** | Proteasome inhibitor | BDR regimen |
| **Venetoclax** | BCL-2 inhibitor | Emerging option |

### Key Considerations
- **IgM flare** with rituximab - avoid if IgM >4000 or symptoms
- **CXCR4 mutations** - slower response to BTK inhibitors
- **MYD88 wild-type** - consider non-WM diagnosis, poor BTK response

## Group G: Question Format/Quality (15 fields)

### 56. cme_outcome_level
**Values:** `"3 - Knowledge"` or `"4 - Competence"`

**CRITICAL RULE:** If the clinical vignette is NOT REQUIRED to answer the question, it's **Level 3 (Knowledge)**, not Level 4.

**Test:** Could someone answer this question correctly WITHOUT reading the patient details?
- YES → `"3 - Knowledge"` (the vignette is just context/decoration)
- NO → `"4 - Competence"` (must apply knowledge to the specific patient)

**WM examples:**
- "What mutation predicts BTK inhibitor response in WM?" → **3 - Knowledge**
- "Patient with WM, IgM 5000, considering rituximab. What concern?" → **4 - Competence**
- "What are cardiac toxicities of ibrutinib?" → **3 - Knowledge**

### 57-60. Format fields
- `data_response_type`: "Numeric", "Qualitative", "Comparative", "Boolean"
- `stem_type`: "Clinical vignette", "Direct question", "Incomplete statement"
- `lead_in_type`: "Standard", "Negative (EXCEPT/NOT)", "Best answer"
- `answer_format`: "Single best", "Compound (A+B)", "All of the above", "None of the above"

### 61. answer_length_pattern
**Values:** "Uniform", "Variable", "Correct longest", "Correct shortest"

### answer_length_pattern Criteria (CRITICAL - Common Error)

**CRITICAL RULE:** If `distractor_homogeneity` = "Homogeneous", then `answer_length_pattern` is almost always "Uniform".

Tag "Variable" with Homogeneous distractors ONLY when one answer contains **additional clinical/educational content** the others lack - NOT when length differs due to:
- Grammatical phrasing
- Compound answers ("Both A and B")
- Minor word count differences

**"Uniform"** = All options are CONCEPTUALLY similar in complexity:
- All single drug names: "Ibrutinib" / "Zanubrutinib" / "Acalabrutinib" → **Uniform**

**"Variable"** = Options differ in TYPE or COMPLEXITY with different amounts of clinical detail

**"Correct longest/shortest"** = The CORRECT answer is noticeably longer/shorter than ALL distractors

### 62. distractor_homogeneity

### distractor_homogeneity Criteria (CRITICAL - Common Error)

**"Homogeneous"** = All distractors are the SAME TYPE of answer:
- All drug names → **Homogeneous**
- All clinical actions → **Homogeneous**

**CRITICAL:** Implausible distractors are still the same TYPE. Tag `flaw_implausible_distractor: true` but keep `distractor_homogeneity: "Homogeneous"` if all options are drugs/treatments.

**"Heterogeneous"** = Distractors are DIFFERENT TYPES of answers:
- Treatment + Safety + Mechanism mixed → **Heterogeneous**
