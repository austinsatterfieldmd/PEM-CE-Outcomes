## CRITICAL: Compound Answer Handling

When the correct answer references options by letter/number (e.g., "B or C", "A and C", "Options 1 and 3"):

1. **DO NOT assume option order matches the original question** - The options you receive may be in a different order than the actual test question.

2. **Deduce the correct answer from clinical reasoning** - Read the substance of each option and determine which ones make sense as the correct answer based on the clinical scenario.

3. **Tag based on the deduced correct content** - Once you identify what the compound answer actually refers to, tag those treatments/concepts.

**Example (CLL):**
- **Stem:** 71-year-old man with treatment-naive CLL, unmutated IGHV, del(13q), no TP53 abnormality. On DOAC for atrial fibrillation, prefers minimal cardiac toxicity. What is the most appropriate frontline treatment?
- **Correct answer:** "B or C"

**How to deduce:** The patient wants to avoid cardiac toxicity and has AFib. Ibrutinib has higher AFib risk than other BTKis. Therefore "B or C" most likely refers to:
- Acalabrutinib-based regimen (2nd-gen BTKi, lower cardiac risk)
- Venetoclax-obinutuzumab (no BTKi cardiac risk)

**Tag accordingly:** treatment_1: Acalabrutinib, treatment_2: Venetoclax, treatment_3: Obinutuzumab

## BTK INHIBITOR (BTKi) CLASSIFICATION RULES (CRITICAL)

**NOTE:** This table lists COMMON drugs for reference. It is NOT exhaustive. If a question mentions a drug, drug class, or mechanism not listed here, STILL tag it using the appropriate fields (treatment_1-5, drug_class_1-3, drug_target_1-3). Always tag what the question discusses, even if not in this table.

BTK inhibitors are important in MCL treatment. **Classify by binding mechanism** (same rules as CLL).

### Covalent vs Non-Covalent BTKi

| Mechanism | Binding | Key Feature | Examples |
|-----------|---------|-------------|----------|
| **Covalent BTKi** | Irreversible (binds Cys481) | Susceptible to C481S resistance | ibrutinib, acalabrutinib, zanubrutinib |
| **Non-covalent BTKi** | Reversible | Active against C481S mutants | pirtobrutinib |

**For any BTK inhibitor encountered, determine if it is covalent or non-covalent. Look up unfamiliar agents to classify correctly.**

**Known covalent BTKi:**
- `"Ibrutinib"` - First approved BTKi in MCL
- `"Acalabrutinib"` - Second-generation, more selective
- `"Zanubrutinib"` - Second-generation, more selective

**Known non-covalent BTKi:**
- `"Pirtobrutinib"` - For BTKi-refractory MCL (BRUIN-MCL)

**Context-based inference:**
- "After ibrutinib resistance" or "C481S mutation" → likely pirtobrutinib (non-covalent)
- "First-line BTKi" or "with bendamustine-rituximab" (SHINE) → covalent BTKi
- "BTKi-refractory" → consider non-covalent BTKi

### answer_length_pattern Criteria

**DEFAULT: If `distractor_homogeneity` = "Homogeneous", ALWAYS tag `answer_length_pattern` = "Uniform".**

Override to "Variable" or "Correct longest/shortest" ONLY when the length difference is EGREGIOUS — e.g., one answer is a single word and another is a full sentence with additional clinical detail.

Do NOT tag "Variable" just because:
- One option has a few extra words
- Compound answers are longer ("Both A and B" vs "A only")
- Drug name lengths differ

**When in doubt with Homogeneous distractors → tag "Uniform".**

Tag "Variable" with Homogeneous distractors ONLY when one answer contains **additional clinical/educational content** the others lack - NOT when length differs due to:
- Grammatical phrasing ("The drug ibrutinib" vs "Ibrutinib")
- Compound answers ("Both A and B", "A and C", "All of the above")
- Minor word count differences

**Example - UNIFORM (despite length difference):**
- "Acalabrutinib/obinutuzumab only"
- "Acalabrutinib monotherapy only"
- "Both acalabrutinib monotherapy and acalabrutinib/obinutuzumab"
→ Third answer is longer but just combines options, no new clinical detail = **Uniform**

**"Uniform"** = All options are CONCEPTUALLY similar in complexity:
- All single drug names: "Ibrutinib" / "Acalabrutinib" / "Venetoclax" → **Uniform** (even if character counts differ)
- All short clinical phrases: "Continue BTKi" / "Switch to CAR-T" / "Add venetoclax" → **Uniform**

**"Variable"** = Options differ in TYPE or COMPLEXITY AND contain different amounts of clinical detail:
- Mix of single words and detailed explanations: "Ibrutinib" / "Continue current therapy and monitor for progression" → **Variable**
- One answer has extra clinical context others lack: "CAR-T" / "Add venetoclax after achieving partial response with monitoring" → **Variable**

**"Correct longest/shortest"** = The CORRECT answer is noticeably longer/shorter than ALL distractors:
- Correct answer is a detailed explanation while distractors are single words → **Correct longest**
- Correct answer is a single drug while distractors are multi-drug regimens → **Correct shortest**

**IMPORTANT:** If answer lengths ARE variable AND the correct answer is the longest → tag `"Correct longest"` (not just "Variable"). Similarly for shortest. These are more specific than "Variable" and should be used when applicable.

### distractor_homogeneity Criteria

**"Homogeneous"** = All distractors are the SAME TYPE of answer:
- All drug names: "Ibrutinib" / "Acalabrutinib" / "Brexu-cel" / "Pirtobrutinib" → **Homogeneous**
- All clinical actions: "Proceed to ASCT" / "Continue BTKi" / "Switch therapy" / "Observe" → **Homogeneous**
- All trial names: "ZUMA-2" / "SHINE" / "SYMPATICO" / "BRUIN-MCL" → **Homogeneous**
- Mix of single drugs and combos: "Ibrutinib" / "BR (bendamustine-rituximab)" / "Venetoclax" / "Lenalidomide" → **Homogeneous** (all are treatment options)

**CRITICAL:** Implausible distractors are still the same TYPE. Tag `flaw_implausible_distractor: true` but keep `distractor_homogeneity: "Homogeneous"` if all options are drugs/treatments.

**"Heterogeneous"** = Distractors are DIFFERENT TYPES of answers:
- Treatment + Prognosis + Testing: "Ibrutinib" / "Poor prognosis with TP53" / "Check Ki-67" → **Heterogeneous**
- Drug + Trial + Biomarker: "Venetoclax" / "ZUMA-2 trial" / "C481S mutation" → **Heterogeneous**
- Mix of domains: "Switch to pirtobrutinib" / "Check for C481S" / "Refer for ASCT" → **Heterogeneous**

## Important Guidelines for MCL

1. **Blastoid/Pleomorphic = aggressive** - Always tag these morphologic variants; they have worse outcomes
2. **MIPI is the standard risk score** - Use "MIPI high", "MIPI low", etc.
3. **BTKi class distinction** - Covalent vs non-covalent is clinically meaningful (C481S resistance)
4. **Brexu-cel is the MCL CAR-T** - Different from axi-cel (DLBCL); ZUMA-2 trial
5. **Age/fitness affects treatment** - Young/fit → intensive + ASCT; Older/unfit → BR or BTKi-based
6. **TP53 mutation = poor prognosis** - Important prognostic marker
7. **Indolent MCL exists** - Rare variant where watch-and-wait may be appropriate
8. **Tag from CORRECT answer only** - Don't tag distractors
   - **Standard questions ("Which of the following...")**: Tag ONLY entities from the correct answer
   - **Negative lead-ins ("All EXCEPT", "Which is NOT")**: Tag the INCORRECT options (distractors) instead, as they represent the true/factual statements. The correct answer is the exception and should NOT be tagged as a treatment/biomarker.
   - **Compound answers ("A and B", "All of the above")**: Tag all components that make up the correct answer
