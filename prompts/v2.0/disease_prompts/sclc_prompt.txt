- "What prophylaxis is needed before starting platinum-etoposide?" → **Prophylaxis** (prevention)

### 2. disease_stage
**REQUIRED FOR SCLC** - Tag the staging system when mentioned or clearly inferable. SCLC uses a unique staging system.

**⚠️ CRITICAL: SCLC uses LIMITED-STAGE vs EXTENSIVE-STAGE, NOT TNM staging**

**Valid values:**
- `"Limited-stage"` - Disease confined to one hemithorax and regional lymph nodes (ipsilateral hilar, mediastinal, supraclavicular) that can be encompassed in a single radiation portal
- `"Extensive-stage"` - Disease beyond limited-stage; includes contralateral hilar/supraclavicular nodes, pleural/pericardial effusion, distant metastases

**When to tag:**
- Staging explicitly mentioned ("Limited-stage SCLC", "Extensive-stage disease", "ES-SCLC", "LS-SCLC")
- Clinical scenario clearly indicates stage:
  - Brain metastases → Extensive-stage
  - Liver metastases → Extensive-stage
  - Malignant pleural effusion → Extensive-stage
  - Disease confined to chest with ipsilateral nodes → Limited-stage

**When null is appropriate:**
- General questions about drug mechanisms without stage context
- Questions applicable across both stages (rare in SCLC due to different treatment paradigms)

**INFERENCE GUIDANCE:**
- **TRY TO INFER disease stage when possible** from clinical context
- Brain metastases, liver metastases, distant metastases → Extensive-stage
- Concurrent chemoradiation, radiation portal encompassing disease → Limited-stage
- First-line immunotherapy combinations (durvalumab, atezolizumab) → Extensive-stage (standard of care)
- If uncertain or ambiguous, use null

**CRITICAL - AVOID OVERLAP:**
- **Disease stage goes in `disease_stage` field ONLY**
- **DO NOT tag "Limited-stage" or "Extensive-stage" in biomarker fields**
- **DO NOT tag "Limited-stage" or "Extensive-stage" in disease_type fields**

**Examples:**
- "Patient with newly diagnosed SCLC with brain metastases" → disease_stage: "Extensive-stage" (brain mets = distant disease)
- "Limited-stage SCLC receiving concurrent chemoradiation" → disease_stage: "Limited-stage"
- "SCLC confined to right lung and ipsilateral mediastinal nodes" → disease_stage: "Limited-stage" (can be encompassed in radiation portal)
- "Bilateral hilar lymphadenopathy with SCLC" → disease_stage: "Extensive-stage" (contralateral hilar nodes = extensive)

**CRITICAL: SCLC is NOT routinely subtyped by molecular markers**

### 5. treatment_line
**OPTIONAL** - Can be null for limited-stage SCLC. More important for extensive-stage SCLC.

**⚠️ CRITICAL: Treatment line rules differ by disease stage in SCLC**

**Valid values:**

**Limited-Stage SCLC (concurrent chemoradiation):**
- `null` - **PREFERRED for limited-stage concurrent chemoradiation** (this is definitive treatment, not a "line" of therapy)
- `"Consolidation"` - PCI after chemoradiation (prophylactic, NOT a treatment line)

**Extensive-Stage SCLC (systemic therapy):**
- `"1L"` - First-line systemic therapy (platinum-etoposide ± immunotherapy)
- `"Maintenance"` - Continuation immunotherapy after completing platinum-etoposide (durvalumab, atezolizumab)
- `"2L+"` - Second-line or later therapy (topotecan, lurbinectedin, tarlatamab)

**⚠️ CRITICAL VALIDATION - Common Errors to Avoid:**
- **"Newly diagnosed" is NOT a valid treatment_line value**
  - If question mentions "newly diagnosed ES-SCLC starting carboplatin-etoposide-atezolizumab" → treatment_line: `"1L"` (NOT "Newly diagnosed")
  - If question mentions "newly diagnosed LS-SCLC starting concurrent chemoradiation" → treatment_line: `null` (NOT "Newly diagnosed")
- **Treatment lines describe SEQUENCE, not diagnosis timing**
- **PCI is NOT a treatment line** - PCI is radiation prophylaxis, tag as treatment_line: `"Consolidation"` only if explicitly described as consolidation

**When to infer:**
- Trial context: IMpower133, CASPIAN → 1L extensive-stage; DeLLphi-300, ATLANTIS → 2L+
- Keywords: "newly diagnosed ES-SCLC", "first-line therapy" → 1L; "after progression on platinum", "post-platinum" → 2L+
- Drug context: Atezolizumab or durvalumab with chemo → 1L; Lurbinectedin or tarlatamab alone → 2L+

**When null is appropriate:**
- Limited-stage SCLC receiving concurrent chemoradiation (this is definitive treatment)
- MOA questions without treatment line context
- General biomarker questions
- Safety profile questions without specific line mentioned

**Examples:**
- "Newly diagnosed extensive-stage SCLC starting carboplatin-etoposide-durvalumab" → treatment_line: "1L"
- "Patient with ES-SCLC completed 4 cycles carboplatin-etoposide-atezolizumab, now continuing atezolizumab" → treatment_line: "Maintenance"
- "Limited-stage SCLC receiving concurrent carboplatin-etoposide and thoracic radiation" → treatment_line: null
- "Patient with LS-SCLC after completing chemoradiation, now receiving PCI" → treatment_line: "Consolidation" (or null if PCI is just prophylaxis)
- "SCLC relapse 4 months after completing platinum-etoposide, starting lurbinectedin" → treatment_line: "2L+"

### 59. cme_outcome_level
**REQUIRED** - Every question must have a CME level. Never null.

**Valid values:**
- `"3 - Knowledge"` - Tests recall of facts, definitions, mechanisms
- `"4 - Competence"` - Tests application to clinical scenarios

**CRITICAL RULE:** If the clinical vignette is NOT REQUIRED to answer the question, it's **Level 3 (Knowledge)**, not Level 4.

**Examples:**

### 63. answer_format
**REQUIRED** - Every question must have an answer format. Never null.

**Valid values:**
- `"Single best"` - One correct answer (most common)
- `"Compound (A+B)"` - Combination answer (e.g., "A and C")
- `"All of above"` - All options are correct
- `"None of above"` - None of the options are correct

**⚠️ CRITICAL: Only tag items from the QUESTION STEM and CORRECT ANSWER**

Do NOT tag treatments, biomarkers, trials, or other entities mentioned ONLY in incorrect/distractor answers.

**When to consider incorrect answers:**
- `"Compound (A+B)"` format → You may need to evaluate multiple options being compared
- `"All of above"` / `"None of above"` → Consider all options to determine correctness
- `"Single best"` → **ONLY tag items from question stem + correct answer**

**Why this matters:** Distractors test whether learners can distinguish correct from incorrect information. Tagging distractors pollutes the dataset with drugs/biomarkers that are NOT actually the focus of the question.

**Examples:**
- Q: "Which immunotherapy is approved for ES-SCLC?" Correct: "Durvalumab", Wrong: "Pembrolizumab"
  - Tag: `treatment_1: "Durvalumab"` (NOT pembrolizumab - it's a distractor)
- Q: "Which of the following are true? A and C" (Compound format)
  - Tag: Items from A, B, C, D as needed to capture what's being evaluated

### 66. answer_length_pattern
**REQUIRED** - Every question must have an answer length pattern. Never null.

**Valid values:**
- `"Uniform"` - All answer options approximately same length
- `"Variable"` - Mix of short and long answers
- `"Correct longest"` - Correct answer is noticeably longest
- `"Correct shortest"` - Correct answer is noticeably shortest

**⚠️ CRITICAL DEFAULT RULE:**
If `distractor_homogeneity` = "Homogeneous", you MUST tag `answer_length_pattern` = "Uniform" UNLESS the length difference is EGREGIOUS (see examples below).

**Rule of thumb:** If you're debating whether it's "Variable", it's probably "Uniform". Only use "Variable" when the difference is obvious and extreme.

**When to override to "Variable":**
Override ONLY when there is an EXTREME disparity in answer length or detail level:
- ✓ One option is a single word ("Durvalumab") and another is a full clinical sentence with multiple concepts ("Continue carboplatin-etoposide with close monitoring for progression every 3 cycles")
- ✓ One option is 3-4 words and another is 15+ words with multiple distinct clinical actions

**When to keep "Uniform" despite minor length differences:**
- ✗ All options are treatment regimens with similar structure, even if one has slightly more detail (e.g., "Carboplatin-etoposide-durvalumab" vs "Topotecan")
- ✗ All options are clinical management choices (e.g., "Continue therapy", "Switch treatment", "Supportive care"), even if one is 2-3 words longer

**"Uniform"** = All options are CONCEPTUALLY similar in complexity:
- All single drug names: "Durvalumab" / "Atezolizumab" / "Topotecan" → **Uniform**
- All short phrases: "Continue therapy" / "Switch treatment" / "Supportive care" → **Uniform**
- All treatment regimens: "Carboplatin-etoposide-durvalumab" / "Carboplatin-etoposide-atezolizumab" / "Topotecan monotherapy" → **Uniform**

**"Variable"** = Options differ in TYPE or COMPLEXITY with EGREGIOUS disparity:
- Mix of single words and detailed explanations: "Durvalumab" / "Continue current carboplatin-etoposide regimen and monitor for progression with serial imaging every 2 cycles" → **Variable**

**"Correct longest/shortest"** = The CORRECT answer is noticeably longer/shorter than ALL distractors

---

### 67. distractor_homogeneity
**REQUIRED** - Every question must have a distractor homogeneity rating. Never null.

**Valid values:**
- `"Homogeneous"` - All distractors from same category
- `"Heterogeneous"` - Mixed distractor types

**"Homogeneous"** = All distractors are the SAME TYPE:
- All drug names: "Durvalumab" / "Atezolizumab" / "Topotecan" / "Lurbinectedin" → **Homogeneous**
- All clinical actions: "Continue therapy" / "Dose reduce" / "Discontinue" → **Homogeneous**
- All trial names: "IMpower133" / "CASPIAN" / "KEYNOTE-604" → **Homogeneous**

**"Heterogeneous"** = Distractors are DIFFERENT TYPES:

### 68-73. flaw_absolute_terms, flaw_grammatical_cue, flaw_implausible_distractor, flaw_clang_association, flaw_convergence_vulnerability, flaw_double_negative

**Item-writing flaws to detect:**

**flaw_absolute_terms** (`true` / `false`):

**flaw_grammatical_cue** (`true` / `false`):

**flaw_convergence_vulnerability** (`true` / `false`):
- Multiple distractors can be eliminated by same reasoning

**flaw_double_negative** (`true` / `false`):
- Double negative phrasing (confusing)

**Default for most well-written questions:** All flaws should be `false`

## Summary: Key Takeaways

1. **Limited-stage vs Extensive-stage is EVERYTHING** - this goes in disease_stage (NOT disease_type, NOT biomarker)
2. **DO NOT tag platinum-etoposide backbone** - carboplatin, cisplatin, etoposide are NOT fundable
3. **TAG immunotherapy** - atezolizumab, durvalumab ARE fundable
4. **PCI is NOT a drug treatment** - it's radiation prophylaxis, NOT tagged in treatment fields
5. **Brain metastases = Extensive-stage** - tag in metastatic_site field, NOT biomarker
6. **SCLC has limited biomarkers** - DLL3, SLFN11, PD-L1 (rare); most questions have biomarker = null
7. **Treatment line for limited-stage = null or Consolidation** - limited-stage gets chemoradiation (not "lines" of therapy)
8. **Platinum-resistant = relapse <90 days** - platinum-sensitive = relapse ≥90 days
9. **Do NOT tag NSCLC biomarkers** - no EGFR, ALK, ROS1, KRAS in SCLC
10. **Tarlatamab targets DLL3** - novel bispecific T-cell engager for relapsed SCLC
