### Secondary Immunodeficiency (SID) Questions — Flag as NON-ONCOLOGY

Patients with DLBCL may develop secondary immunodeficiency from disease or treatment. Questions about **managing infections** belong to **Infectious Disease/Immunology**, NOT oncology.

**Red flags that suggest NON-ONCOLOGY (SID):**
1. Activity name contains "SID", "Secondary Immunodeficiency", or "Immunodeficiency"
2. Patient has DLBCL BUT the question asks about:
   - Infection diagnosis/management (PJP, fungal, viral reactivation)
   - Vaccination in immunocompromised patients
   - Post-CAR-T infection management (not CRS/ICANS)
3. Answer choices focus on infectious disease concepts, NOT lymphoma treatment

**If you detect this pattern:**
- Tag `topic: "Non-oncology - SID"` to flag for manual review

**True oncology questions:**

### 15. drug_class_1 through drug_class_3
Therapeutic class of the fundable drug(s).

**CRITICAL — Use GENERIC drug class names.** The target (CD19, CD20, CD79b, etc.) goes in `drug_target_1-3`, NOT in drug_class.

**DLBCL-relevant classes:**
- `"CAR-T therapy"` - axi-cel, tisa-cel, liso-cel
- `"Bispecific antibody"` - glofitamab, epcoritamab, mosunetuzumab
- `"Antibody-drug conjugate (ADC)"` - polatuzumab, loncastuximab
- `"Monoclonal antibody"` - rituximab, obinutuzumab, tafasitamab (distinguish by drug_target)
- `"IMiD"` - lenalidomide (target = Cereblon)
- `"XPO1 inhibitor"` - selinexor

**Monoclonal antibody rule:**
- ALL monoclonal antibodies use `drug_class: "Monoclonal antibody"` — distinguish by drug_target
- Rituximab, obinutuzumab: `drug_class: "Monoclonal antibody"` + `drug_target: "CD20"`
- Tafasitamab: `drug_class: "Monoclonal antibody"` + `drug_target: "CD19"`
- This enables analysis of mAbs as a class while distinguishing mechanisms by target

**Examples:**
- Correct: `drug_class_1: "Monoclonal antibody"` + `drug_target_1: "CD20"` (for rituximab)
- Incorrect: `drug_class_1: "Anti-CD20 antibody"` (target embedded in class name)
- Correct: `drug_class_1: "IMiD"` + `drug_target_1: "Cereblon"` (for lenalidomide)
- Incorrect: `drug_class_1: "IMiD"` with no drug_target (missing Cereblon)

## Key Rules for Output

1. **All 70 fields must be present** - Never omit a field; use `null` for inapplicable fields
2. **Exact canonical values** - Use values exactly as specified
3. **Valid JSON** - Proper formatting
4. **One topic only** - Topic is required; never null
5. **Boolean flaw fields** - Use `true` or `false` (not strings, not null)
6. **CAR-T product specificity** - Always try to identify axi-cel, tisa-cel, or liso-cel
7. **Bispecific product specificity** - Tag specific agent, not generic "bispecific"
8. **IPI for risk** - Use "IPI high", "IPI low", etc. (not just "High-risk")
9. **Tag from correct answer** - Not distractors
